    3. Programming insulin delivery settings (18 min)

    3. Programming insulin delivery settings (18 min)¶

    Welcome to this lesson on programming insulin delivery settings with the Equil Pump.

    In this video, you’ll learn how to set up and edit basal programs, apply a temporary basal rate when needed, and deliver boluses using manual, preset, extended, and quick-bolus options. We’ll also show you how to use the Bolus Calculator to support dosing decisions and how to suspend and resume insulin delivery safely. Finally, we’ll review practical do’s and don’ts to help you adjust settings correctly and avoid common errors. Let’s dive in.

    ¶

    3.1 Setting a basal rate¶

    From the Home Screen, press the Basal function button. You will see three basal program tabs, out of the seven programs the pump can run. You can navigate between them using the touchscreen. Here, you are viewing data for a lighter basal program. The top row displays the total basal amount, the program status — that is, whether it is active or not — and a list of basal program periods with their corresponding values and times. Swipe down in this area to scroll and view additional entries. All values in this list will always be lower than the Maximum Basal Rate. At the bottom of the screen, you’ll find the basal program action buttons, which allow you to activate a program or change its settings.

    If you rotate the PDA to landscape orientation, the basal program graph mode is activated. In this view, the vertical axis represents the basal rate, while. Because the basal rate is currently set to a fixed value and no program periods have been defined, it appears as a straight line. Once a program is configured, the different basal periods will be shown both as a list and as a graphical display. Basal rates often vary throughout the day depending on daily activities. Since activity patterns can differ across days (such as workdays versus weekends, or fasting days versus regular days), you may need to create and activate different basal programs, switching between them as needed.

    The Equil Pump also offers a quick basal program setup option, making it easier and faster to edit basal programs. To begin, enter Edit Mode by selecting the top row below the program name and tapping the Edit icon. The Basal Initial Setup window will appear. Use the plus and minus buttons to adjust the total daily basal amount, then choose how many time segments you want — either 24 or 6. Based on these settings, the system will automatically assign basal time periods to the program. To modify any time period, tap its Edit icon and, in the pop-up window, set the start time, end time, and basal rate. Keep in mind that changing individual time periods may slightly affect the total daily basal amount. You can also delete a time period by selecting the Cancel icon; in this case, the period will revert to the base basal rate. Note that the PDA may automatically merge adjacent time periods that have the same basal rate.

    Basal programs can also be edited in graphical mode. Tap and hold a section of the graph to select a time period; the selected area will appear darker. You can move between periods using the left and right arrows, or by using the touchscreen. As in portrait mode, this icon opens the Basal Rate Settings window. Selecting the Cancel icon removes the selected time period and returns it to the base basal value, which you’ll see reflected immediately on the chart. This icon exits Edit Mode and cancels any changes, while selecting the Save option exits Edit Mode and saves your settings.

    When you are not in Edit Mode, you can use the touchscreen to move between different basal programs. Renaming a basal program is done in portrait mode. Enter Edit Mode, tap Change Basal Program Name, and enter a new name in the dialog box. After renaming, tap Save Settings. The program can then be activated by selecting Activate and confirming by tapping OK. The active basal program’s name will appear on the Home Screen. If you edit a program that is already active, an Update Pump dialog box will appear after saving. Select Cancel to discard the changes, or OK to apply the updates to the pump’s current basal program.

    ¶

    3.2 Setting a temporary basal rate¶

    Before you can use a temporary basal rate, the feature must be enabled. You can activate it either from page 2 of the Settings Wizard or by going to Settings > Insulin Delivery Settings > Temporary Basal Rate.

    The Temporary Basal option can be used only within the currently active basal program. After the program is activated, the same activation button changes to Temporary Basal Rate. The button appears green when the Temporary Basal option is enabled. If it is not enabled, the button will appear gray.

    Press the Temporary Basal Rate button to open the settings dialog box. Enter the Temporary Basal rate value in units per hour, based on the settings you selected earlier. Then set the duration and tap OK. The PDA will send the command to the pump to begin delivering the updated rate, and the Home Screen will display the Temporary Basal status.

    3.3 Delivering a bolus¶

    There are several methods to administer a bolus. The first is a manual bolus. From the Home Screen, tap the Bolus button to open the Bolus screen. In the upper-left corner, you’ll see a green Bolus button. Initially, the bolus amount is zero, and the Start button is greyed out. Tap the Bolus button to open a dialog box and enter the desired bolus amount, then press OK. The insulin amount will appear in the box, and the Start button will turn green. Tap Start to begin delivery. A “Connecting to Pump” animation will appear, followed by a confirmation window showing your last bolus amount and time. Tap OK to proceed. The PDA will return to the Home Screen and display the bolus delivery status at the previously configured rate. The Bolus button will remain inactive until the bolus has been fully delivered.

    The Bolus screen also includes three programmable preset bolus buttons. Tap the Edit icon to open the Preset dialog box. Adjust the dose using the plus and minus buttons, assign a name, and tap OK. When you tap the left side of a preset button, its preset value appears in the bolus amount box on the left, along with a small arrow linking the preset to the selected dose. The Start button turns green, and you can proceed as before. Preset boluses are especially useful for fast-digesting foods, as the full dose is delivered immediately.

    For meals that digest more slowly, you can use the Extended Bolus option. Extended boluses are delivered in two parts: Amount Now, which is delivered immediately, and the remaining amount, which is delivered over a set period of time. First, enter the total bolus amount as usual. Before starting, tap Extend Bolus to open the Extended Bolus window. Under Amount Now, enter the portion to be delivered immediately, using absolute units as defined in your settings. Then set the duration over which the remaining insulin will be delivered. Tap OK, then Start, and follow the same confirmation steps as before.

    While delivery is in progress, the PDA returns to the Home Screen and displays the Extended Bolus status, including how much insulin has been delivered from each part. A countdown timer shows how much time remains for the extended portion. For high-fat meals, an extended bolus without an immediate dose may be appropriate. In this case, set Amount Now to zero. No insulin will be delivered immediately, and the full dose will be spread over a period ranging from 30 minutes to 8 hours.

    The Equil Pump also includes a Quick Bolus feature. This allows you to administer a bolus directly using the Bolus button on the patch pump, without using the PDA. This can be useful if you are away from your PDA or prefer more privacy. Press and hold the Bolus button on the patch pump. An audible alert indicates that Quick Bolus mode is active. Each additional press increases the bolus by a fixed increment. The total bolus equals the number of button presses multiplied by the Quick Bolus Increment. For example, if the increment is 2 units and you need a 10-unit bolus, press the button five times. Each press is confirmed by a beep, with the tone increasing in pitch up to five tones to help you count presses. The total Quick Bolus dose cannot exceed the preset Maximum Bolus value. After entering the desired dose, wait three seconds. The pump will replay the tones to confirm the dose. If correct, press the button once more to start delivery. If the tones are incorrect, wait more than 10 seconds and the Quick Bolus will be cancelled automatically. You can also deliver a Quick Bolus using the Enter button on the PDA instead of the pump button.

    In some situations, you may need to cancel a bolus after it has started — for example, if you entered an incorrect dose, miscalculated carbohydrates, entered the wrong blood glucose value, or changed your meal plan. While a bolus is in progress, tap Cancel on the Home Screen. A confirmation window will appear. Tap OK to confirm, and a dialog box will display the amount and time of insulin already delivered. A bolus can also be cancelled directly from the pump by pressing and holding the Quick Bolus button. You will hear an audible confirmation, and the bolus will stop immediately. A dialog box will then show the delivered dose details.

    Be aware that canceling a bolus stops the Total Bolus, Amount Now, and Extended Amount. If you are manually recalculating a new dose, you must account for the insulin already delivered and subtract it from the next dose. If you are using the Bolus Calculator, the cancelled dose is automatically included as Active Insulin and deducted from the new bolus.

    3.4 Using the Bolus Calculator¶

    The Bolus Calculator can suggest a bolus dose after you test your blood glucose level and enter the amount of carbohydrates you will be eating, helping keep your blood glucose within the target range. This feature determines the amount of insulin needed to cover the rise in blood glucose associated with carbohydrate intake as the Meal Bolus. In addition, a Correction Bolus is added to help bring your current blood glucose level back to the target range. And if you have recently taken a bolus, there may still be active insulin in your body, so the Bolus Calculator will automatically subtract the active insulin from the Total Suggested Bolus dose.

    The Bolus Calculator is switched off by default. To activate it from the Home screen, press Settings > Blood Glucose Settings, then turn it on. You’ll see five settings displayed; these values should be adjusted in consultation with your healthcare provider. First is the Target Blood Glucose Range, which sets the default target range. This is a user setting confirmed by the healthcare provider. The Equil Pump also allows you to set different target ranges for different time periods, and the calculator takes the average adjusted value into account. Next is the Carbohydrate Ratio (ICR), which refers to how many carbohydrates are covered per unit of insulin and is used to calculate the Meal Bolus. Because metabolism varies from person to person, set this value under the guidance of your healthcare provider. Note that different carbohydrate ratios can also be set for different time periods. The Insulin Sensitivity Factor (ISF) indicates how much your blood glucose may drop after taking one unit of insulin. Since this value also differs between individuals, consult your healthcare provider, and remember that ISF can be set for different time periods as well. Even if you have recently taken a bolus, you may still have active insulin in your body, as insulin typically remains active for four to six hours, and this value must be determined by your doctor. Finally, Reverse Correction affects how correction boluses are calculated. If it is off, the Bolus Calculator will consider a correction bolus only when your current blood glucose is higher than the target range. If it is on, a correction bolus is always calculated. The default setting is on, but your doctor can change it if necessary. Once you’ve adjusted the settings and understood the inputs, you can use the Bolus Calculator quickly and easily.

    Only two inputs are needed to generate a bolus suggestion: your current blood glucose level and the amount of carbohydrates you will be eating. On the Home screen, select Actions, then click Enter BG Reading to open the manual entry screen. Using the plus and minus buttons, enter your blood glucose reading, select the appropriate tag — Before/After Exercise or Before/After Meal — and press Save. When the dialog box appears, choose OK to save the result. You’ll notice that the Bolus Calculator button changes from grey to green, indicating that it has been activated. The internal PDA blood glucose meter can also activate the Bolus Calculator. If the display is on, insert an Exactive EQ test strip and the blood glucose meter screen will appear. Apply enough blood sample, then the screen will count down five seconds. The test result will be displayed, and the Bolus Calculator button will turn green once a blood glucose reading has been recorded. Note that your blood glucose reading is valid for only ten minutes. If you do not deliver a bolus within 10 minutes of your test, please test again.

    Now, press the Bolus Calculator button. It will ask you, “Will you be eating now?” If you choose Yes, a dialog box will appear asking you to enter the amount of carbohydrates. You can estimate the carbohydrates in your food by reading packaging labels, using carbohydrate-counting books or phone apps, or referring to the PDA’s Food Library. Enter the total amount of carbohydrates you intend to eat and press OK. The next page is the Suggested Bolus screen. At the top, you’ll see the suggested bolus dose. The middle section shows the calculation parameters, and the bottom contains action buttons. The suggested bolus amount appears automatically in the Bolus Amount area. Tapping the number opens a dialog box that allows you to change the value if your physician has advised adjustments, or you can use it as is. The Details button opens a window that explains how the suggested bolus was calculated. By pressing Start, you can administer the bolus dose. Alternatively, you may use the Extended Bolus option, as explained earlier. However, if you choose No in the “Will you be eating now?” dialog box, the Meal Bolus will not be included.

    3.5 Suspending delivery¶

    Occasionally, you may need to temporarily pause insulin delivery using the Suspend function — for example, if you do not need insulin infusion for a short period or you need to remove the pump from the infusion set. From the Home Screen, tap Suspend to stop insulin delivery. A confirmation dialog box will appear; select OK to suspend or Cancel to exit. You may also choose to stop delivery and allow the pump to rewind. Keep in mind that if the pump rewinds, you will need to prime the reservoir again before resuming use.

    Once Suspend mode is activated, the PDA returns to the Home Screen, and the information area will show how long insulin delivery has been paused. The Suspend/Stop button will change to a Resume button.

    To restart insulin delivery, tap Resume. When the confirmation dialog box appears, select OK to resume insulin delivery.

    3.6 Do's & don’ts¶

    • At pump start-up, specific bolus settings are configured using the Settings Wizard, in consultation with your healthcare provider. These settings can be adjusted later from the Home Screen > Settings > Insulin Delivery Settings > Bolus Rate.

    • The normal bolus delivery rate is 3.0 units per minute. This can be reduced to 1.5 units per minute if needed.

    • The Bolus Increment setting determines how much the requested bolus changes each time you press the plus or minus button. It can be set between 0.1 and 2 units.

    • The Maximum Bolus setting defines the largest single bolus dose the pump can deliver. This safety feature helps reduce the risk of overdose due to user error. It is recommended to set this value to the largest bolus you would normally administer. Setting it too high increases the risk of overdosage. You can also configure the Extended Bolus option. By default, this feature is turned off and the corresponding button appears grayed out. Once enabled, you can choose how much insulin is delivered immediately, either in insulin units or as a percentage of the total bolus dose.

    • The final setting is Quick Bolus, which is inactive by default. Select the checkbox next to Quick Bolus to enable it. The Quick Bolus Increment determines how much the bolus increases with each press of the Quick Bolus button.

    • When you open the Basal screen for the first time, no basal program is active. If you activate one of the three basal programs, the previously running basal rate will stop automatically.

    • The Initial Basal Setup provides only an estimate of daily basal needs. For individualised dosing, consult your healthcare provider.

    • You cannot switch between basal programs in Graph Display mode. Return to List Display to select a different basal program.

    • If you edit a basal program that is currently running on the pump, a dialog box will ask whether you want to send the updated program to the pump. Select OK to update and save, or Cancel to return to the Basal Edit screen.

    • The maximum number of increments for a Quick Bolus is 20.

    • To cancel a Quick Bolus using the PDA, press the Back button, or wait 10 seconds for the system to return to the Lock Screen.

    • If there is no activity for 10 seconds, the pump will exit Quick Bolus mode automatically. To cancel a Quick Bolus using the pump, simply wait 10 seconds for the system to exit Quick Bolus mode.
