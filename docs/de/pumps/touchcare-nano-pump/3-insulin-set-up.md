## **<u>3. Insulin einrichten</u>**

Willkommen zu dem Teil, in dem wir Ihre Insulinabgabe einrichten - keine Sorge, es ist einfacher, als es klingt! Wir zeigen Ihnen, wie Sie Ihre Basalinsulinabgabe programmieren, eine temporäre Basalrate einstellen, Insulin aussetzen, einen Insulinbolus abgeben und geben Ihnen einige einfache Tipps, damit alles reibungslos abläuft.

Wir werden Ihnen immer zuerst zeigen, wie Sie es mit dem PDM und dann mit der EasyPatch-App machen.

### 3.1 Programmieren Ihres Basalinsulins

Wenn Sie den PDM verwenden, gehen Sie zum Hauptmenü und tippen Sie auf "Einstellungen", dann auf "Insulinpumpe" und dann auf "Basaleinstellung". Tippen Sie auf "Basalrate bearbeiten", und wählen Sie "Standard", um das Muster zu bearbeiten.

Hinweis: Das aktive Basalratenmuster ist mit einem Häkchen versehen.

Der Bearbeitungsbildschirm wird angezeigt. Tippen Sie auf +Zeitabschnitt hinzufügen, um einen neuen Abschnitt zu erstellen und hinzuzufügen. Geben Sie die Endzeit für dieses Segment ein. Tippen Sie dann auf “Fertig”.  Ihr erstes Zeitsegment beginnt um Mitternacht. Das letzte Basalsegment endet immer um 24:00 (oder 12:00 Uhr).

Tippen Sie auf jeden Abschnitt, um die Zahlen einzugeben. Wenn Sie mit den Einstellungen zufrieden sind, tippen Sie auf "Speichern". Wenn Sie zu verschiedenen Tageszeiten unterschiedliche Einstellungen benötigen, tippen Sie auf das Plus-Symbol, um weitere Segmente hinzuzufügen.

Wählen Sie zum Löschen ein Segment aus, schieben Sie es nach links und eine Schaltfläche zum Löschen wird angezeigt. Tippen Sie auf “Löschen”, um das ausgewählte Segment zu löschen. Schieben Sie zurück nach rechts, um Löschen zu schließen. Das letzte Segment kann nicht gelöscht werden, und die Startzeit aller Segmente kann nicht geändert werden.

Wenn du die EasyPatch-App verwendest, kannst du deine Basalrate in deiner EasyPatch-App einrichten, tippen Sie auf die drei Linien in der oberen linken Ecke, um das Hauptmenü aufzurufen. Tippen Sie auf “Einstellungen”. Tippen Sie auf “Insulinpumpe”. Tippen Sie auf “Basalprofil bearbeiten”. Tippen Sie auf “Standard”. Sie können verschiedene Zeitsegmente für den Tag hinzufügen, je nachdem, ob Sie unterschiedliche Insulinmengen zu verschiedenen Zeiten benötigen. Tippen Sie auf “Zeitsegment hinzufügen”, um ein neues Zeitsegment hinzuzufügen. Sie können die Insulinrate für jedes Zeitsegment anpassen, indem Sie auf die blaue Zahl neben dem jeweiligen Zeitsegment tippen. Tippen Sie nun auf “Fertig”. Wenn Sie ein Zeitsegment löschen möchten, streichen Sie einfach mit Ihrem Finger von rechts nach links über das Zeitsegment und tippen Sie dann auf “Löschen”. Wenn Sie die Basalrateneinstellung abgeschlossen haben, tippen Sie auf “Fertig” in der oberen rechten Ecke Ihres Smart-Geräts, um Ihre Basalrate zu speichern.

### 3.2 Einstellen einer temporären Basalrate

Manchmal brauchen Sie vielleicht etwas mehr oder weniger Insulin - zum Beispiel beim Sport oder bei Krankheit. Dazu können Sie eine temporäre Basalrate einstellen.

Wenn Sie den PDM verwenden, um eine temporäre Basalrate auf Ihrer Pumpe abzugeben, schieben Sie, um Ihren PDM zu entsperren. Streichen Sie Ihren Finger von rechts nach links, um das Hauptmenü aufzurufen. Tippen Sie auf “Basal”. Tippen Sie auf “Basaltemperatur”. Entscheiden Sie, ob Sie Ihre Basalrate erhöhen oder verringern möchten. Tippen Sie auf die blaue Zahl neben der Prozentzahl. Wählen Sie nun aus, wie viel Basalinsulin Sie abgeben möchten. Tippen Sie auf “Fertig”. Tippen Sie auf die blaue Zahl neben der Dauer. Wählen Sie nun die Zeit aus, wie lange die temporäre Basalrate laufen soll. Tippen Sie auf “Fertig”. Tippen Sie nun auf “Weiter”. Der PDM zeigt Ihnen eine Aufschlüsselung an, wie die temporäre Basalrate abgegeben wird. Schieben Sie nun, um die temporäre Basalrate zu aktivieren. Die Pumpe aktiviert nun die Abgabe der temporären Basalrate. Dies wird durch einen dunkelgrünen Kreis auf Ihrem Startbildschirm angezeigt, unter dem die temporäre Basalrate steht.

Wenn Sie die EasyPatch-App, zur Abgabe einer temporären Basalrate an Ihre Pumpe verwenden, tippen Sie auf die drei Linien in der oberen linken Ecke, um das Hauptmenü aufzurufen. Tippen Sie auf “Insulinabgabe”. Tippen Sie auf “Basaltemperatur”. Entscheiden Sie, ob Sie Ihre Basalrate erhöhen oder verringern möchten. Wählen Sie nun aus, wie viel Basalinsulin Sie abgeben möchten. Tippen Sie auf “Fertig”. Tippen Sie auf die blaue Zahl neben der Dauer. Wählen Sie nun die Zeit aus, wie lange Ihre temporäre Basalrate laufen soll. Tippen Sie auf “Fertig”. Tippen Sie nun auf “Weiter”. Ihr Smart-Gerät zeigt Ihnen nun eine Aufschlüsselung des Prozentsatzes und der Dauer der temporären Basalrate an. Schieben Sie nun, um die temporäre Basalrate zu aktivieren. Die Pumpe aktiviert nun die Abgabe der temporären Basalrate. Dies wird durch einen dunkelgrünen Kreis auf Ihrem Startbildschirm angezeigt, wobei die temporäre Basalrate unten dargestellt ist.

### 3.3 Unterbrechen der Insulinabgabe

Wenn Sie den PDM verwenden, um die Insulinabgabe an Ihre Pumpe zu unterbrechen, schieben Sie die Taste, um Ihren PDM zu entsperren. Streichen Sie mit Ihrem Finger von rechts nach links, um das Hauptmenü aufzurufen. Klicken Sie auf “Unterbrechen”. Nun können Sie die Zeitspanne auswählen, für die Sie Ihre Pumpe aussetzen möchten. Sie können eine Zeitspanne von 15 Minuten bis zu 2 Stunden in 15-Minuten-Schritten einstellen. Wählen Sie “Fertig”. Wählen Sie “Weiter”. Schieben Sie nun, um die Insulinabgabe zu unterbrechen. Die Pumpe unterbricht nun die Insulinabgabe. Dies wird durch einen roten Kreis auf dem Startbildschirm angezeigt, darunter steht die verbleibende Unterbrechungszeit. Um die Insulinabgabe Ihrer Pumpe wieder aufzunehmen, schieben Sie die Taste, um Ihren PDM zu entsperren. Streichen Sie mit Ihrem Finger von rechts nach links, um das Hauptmenü aufzurufen. Klicken Sie auf “Fortsetzen”. Schieben Sie nun, um die Abgabe Ihres Basalinsulins fortzusetzen. Die Pumpe nimmt nun die Abgabe Ihres Basalinsulins wieder auf. Dies wird durch einen grünen Kreis auf Ihrem Startbildschirm angezeigt, unter dem Ihre Basalabgabe steht.

Wenn Sie die EasyPatch App verwenden, um die Insulinabgabe Ihrer Pumpe zu unterbrechen, tippen Sie unten auf Ihrem Smartgerät auf "Unterbrechen". Nun können Sie die Zeitspanne auswählen, für die Sie Ihre Pumpe aussetzen möchten. Sie können eine Zeitspanne von 15 Minuten bis zu 2 Stunden in 15-Minuten-Schritten einstellen. Wählen Sie “Fertig”. Wählen Sie “Weiter”. Schieben Sie nun, um die Insulinabgabe zu unterbrechen. Tippen Sie nun auf “OK”. Die Pumpe unterbricht nun die Insulinabgabe. Dies wird durch einen roten Kreis auf dem Startbildschirm und die verbleibende Unterbrechungszeit darunter angezeigt. Um die Insulinabgabe Ihrer Pumpe wieder aufzunehmen, tippen Sie unten auf Ihrem Smart-Gerät auf "Fortsetzen". Schieben Sie nun, um die Abgabe Ihres Basalinsulins fortzusetzen. Die Pumpe nimmt nun die Abgabe Ihres Basalinsulins wieder auf. Dies wird durch einen grünen Kreis auf dem Startbildschirm angezeigt, unter dem Ihre Basalabgabe steht.

### 3.4 Abgabe eines Bolus

Bei der TouchCare Nano-Pumpe können Sie sich für einen normalen Bolus, einen verlängerten Bolus oder einen Kombibolus entscheiden - je nachdem, was für Sie besser geeignet ist. Im Einstellungsmenü können Sie auch den Bolus-Rechner aktivieren. Mit dieser Funktion können Sie Ihr Kohlenhydratverhältnis, Ihre Insulinempfindlichkeit, Ihren Zielglukosespiegel und die Dauer der Insulinwirkung einstellen, um den Insulinbedarf für Mahlzeiten und Korrekturen genauer abzuschätzen.

Wenn Sie den PDM verwenden, schieben Sie, um Ihren PDM zu entsperren. Streichen Sie Ihren Finger von rechts nach links, um das Hauptmenü aufzurufen. Tippen Sie auf “Bolus”. Tippen Sie auf “Bolusrechner” und geben Sie Ihren Blutzuckerwert ein, indem Sie auf die drei Striche unter Blutzuckerwert (BG) tippen. Tippen Sie nun auf “Fertig”. Geben Sie Ihre Kohlenhydrate ein, indem Sie auf die Null bei den Kohlenhydraten tippen. Tippen Sie nun auf “Fertig”. Tippen Sie auf “Weiter”. Tippen Sie erneut auf “Weiter”. Schieben Sie nun, um den Bolus abzugeben. Um einen manuellen Bolus auf Ihrer Pumpe abzugeben, schieben Sie, um Ihren PDM zu entsperren. Streichen Sie mit Ihrem Finger von rechts nach links, um das Hauptmenü aufzurufen. Tippen Sie auf “Bolus”. Tippen Sie auf “Manueller Bolus”. Geben Sie die Insulinmenge ein, die Sie verabreichen möchten, indem Sie die drei Nullen antippen, um den Bolus einzustellen. Tippen Sie nun auf Fertig. Tippen Sie auf “Weiter”. Schieben Sie nun, um den Bolus abzugeben. Die Pumpe gibt nun Ihren manuellen Bolus ab.

Wenn Sie die EasyPatch-App verwenden, tippen Sie auf die Bolus-Taste in Ihrer App. Geben Sie Ihren Blutzucker ein, indem Sie auf die beiden Striche neben Blutzuckerwert (BG) tippen. Geben Sie Ihre Kohlenhydrate ein, indem Sie auf die Null neben Kohlenhydrate tippen. Tippen Sie nun auf “Fertig”. Tippen Sie jetzt auf Weiter. Tippen Sie jetzt erneut auf Weiter. Schieben Sie nun, um den Bolus abzugeben. Die Pumpe gibt nun Ihren Bolus ab. Um einen manuellen Bolus an Ihrer Pumpe abzugeben, tippen Sie auf die drei Linien in der oberen linken Ecke, um das Hauptmenü aufzurufen. Tippen Sie auf Insulinabgabe. Tippen Sie auf Manueller Bolus. Geben Sie die Insulinmenge ein, die Sie verabreichen möchten, indem Sie auf die drei Nullen neben der Einstellung tippen. Tippen Sie nun auf Fertig. Tippen Sie jetzt auf Weiter. Schieben Sie nun, um den Bolus abzugeben. Die Pumpe gibt nun den manuellen Bolus ab.

### 3.5 Was man tun und was man nicht tun sollte

- Wenn Sie eine Insulinpumpe verwenden, ist es immer am besten, zur Sicherheit die maximale Basal- und Bolusabgabe einzustellen. Sowohl auf dem PDM als auch auf der EasyPatch-App kann dies über die Einstellungen vorgenommen werden. Die maximale Basalrate sollte mindestens das Zweifache der höchsten aktuellen Basalinsulinabgabe betragen, und der maximale Bolus sollte etwas höher als Ihre übliche größte Dosis eingestellt werden.

- Die TouchCare Nano Pumpe ermöglicht es Ihnen, mit voreingestellten Bolusmengen für Frühstück, Mittagessen, Abendessen und Zwischenmahlzeiten zu arbeiten. Diese können über das Einstellungsmenü voreingestellt werden.

Das war's! Sie können jetzt sicher Insulin programmieren, einstellen und verabreichen. Machen Sie weiter so!
