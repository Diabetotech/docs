## **<u>3. Insulin einrichten</u>**

Willkommen zu unserer ausführlichen Anleitung zur Einrichtung der Insulinabgabe für die Tandem Mobi Insulinpumpe. 

Wir werden uns damit befassen, wie man das Basalinsulin programmiert, eine temporäre Basalrate einstellt und einen Insulinbolus verabreicht. Darüber hinaus werden wir Ihnen einige wichtige Verhaltensregeln vermitteln, die Ihnen helfen, Ihre Erfahrungen mit der Insulinpumpe zu verbessern.

### 3.1 Programmierung des Basalinsulins 

- Um auf Ihr persönliches Profil zuzugreifen, tippen Sie in der Navigationsleiste auf „Einstellungen“, dann auf „Pumpe“, „Persönliche Profile“, „Neues Profil hinzufügen“ und anschließend auf das Feld „Profilname“.

- Geben Sie einen Profilnamen mit bis zu 16 Zeichen ein und tippen Sie auf „Weiter“. Tippen Sie auf „Segment hinzufügen“, um mit der Einstellung der Insulinabgabe zu beginnen. Das erste Zeitsegment beginnt um 12 Uhr.

- Innerhalb jedes Zeitsegments müssen Sie eine Basalrate, einen Korrekturfaktor, ein Kohlenhydratverhältnis und einen Ziel-Blutzuckerwert programmieren. Um Werte für jede Variable einzugeben, tippen Sie auf den Bereich „Zum Einrichten antippen“. Es erscheint eine Zifferntastatur. Nachdem Sie die erste Zahl eingegeben haben, tippen Sie einfach auf das Feld, um den nächsten Wert einzugeben. Tippen Sie auf „Fertig“, um die Tastatur zu schließen. 

- Überprüfen Sie alle Werte, die Sie gerade eingegeben haben, und tippen Sie auf „Speichern“. Oben in der Tandem Mobi-App wird die Meldung „Zeitsegment wurde gespeichert“ angezeigt.

- Wenn Sie weitere Zeitsegmente hinzufügen möchten, tippen Sie auf das Plus-Symbol. Beim Hinzufügen weiterer Zeitsegmente werden alle Einstellungen, die Sie im vorherigen Zeitsegment vorgenommen haben, kopiert und im neuen Segment angezeigt. So können Sie ganz einfach nur die gewünschten Einstellungen anpassen, anstatt sie alle erneut eingeben zu müssen. 

- Tippen Sie auf „Segment hinzufügen” und dann auf „Startzeit”. Geben Sie über die Bildschirmtastatur die gewünschte Startzeit für das Zeitsegment ein und tippen Sie auf „Fertig”. Beachten Sie, dass dieses Segment ab der Startzeit bis 12 Uhr nachts oder bis zum nächsten von Ihnen eingegebenen Segment aktiv ist. 

- Wenn Sie Ihre Einstellungen bestätigt und keine weiteren Zeitabschnitte hinzugefügt haben, tippen Sie auf das Zurück-Symbol, um Ihre Bolus-Einstellungen zu programmieren. Tippen Sie zunächst auf das Feld „Bolus-Einstellungen“, um zu beginnen. Standardmäßig ist die Insulinwirkdauer auf 5 Stunden eingestellt und die Kohlenhydrate sind aktiviert. Wir empfehlen, diese Einstellungen auf den Standardwerten zu belassen. 

- Um einen Zeitabschnitt zu löschen, tippen Sie auf die Startzeit des Zeitabschnitts, den Sie löschen möchten, und tippen Sie auf „Abschnitt löschen“. Tippen Sie zur Bestätigung auf „Ja“.

- Wenn Sie ein vorhandenes Profil umbenennen möchten, tippen Sie in der Navigationsleiste auf „Einstellungen“. Tippen Sie auf „Pumpe“ und dann auf „Persönliche Profile“. Tippen Sie auf den Namen des persönlichen Profils, das Sie umbenennen möchten. Tippen Sie auf „Profiloptionen“ und dann auf „Umbenennen“. 

- Wenn Sie Ihre Einstellungen bestätigt haben, tippen Sie auf „Dashboard“, um zum Startbildschirm zurückzukehren. Wenn Sie weitere persönliche Profile hinzufügen möchten, wiederholen Sie die in diesem Video beschriebenen Schritte. 

- Sie können auch ein bestehendes persönliches Profil duplizieren. Wählen Sie dann „Bearbeiten“, um nur bestimmte Einstellungen zu ändern und ein neues Profil zu erstellen.

Um die Basalinsulinabgabe zu stoppen, tippen Sie in der Navigationsleiste auf „Aktionen“, wählen Sie „Insulin stoppen“ und tippen Sie auf „Ja“. Wenn die Insulinabgabe gestoppt ist, wird auf dem Startbildschirm der Status „Alle Abgaben gestoppt“ angezeigt. Die Basalinsulinabgabe kann auf die gleiche Weise über „Aktionen“ wieder gestartet werden.

### 3.2 Vorübergehende Basal

Um eine temporäre Basalrate zu starten, navigieren Sie in der Navigationsleiste zu „Aktionen” und wählen Sie „Temp Rate” (temporäre Rate). Hier können Sie den Prozentsatz und die Dauer Ihrer Wahl einstellen. Tippen Sie in der Navigationsleiste auf „Dashboard“, um den Dashboard-Bildschirm anzuzeigen und das Symbol zu bestätigen, das anzeigt, dass eine temporäre Rate aktiv ist. Ein „T“ in einem gelben Kästchen bedeutet, dass die temporäre Rate aktiv ist, während ein „T“ in einem roten Kästchen bedeutet, dass eine temporäre Rate von 0 u/h aktiv ist. Sie können die temporäre Basalrate jederzeit manuell deaktivieren, wodurch das „T“ wieder auf die aktuellen Systemeinstellungen zurückgesetzt wird.

3.3 Bolusgabe

Die Verabreichung eines Bolus ist ganz einfach. Drücken Sie auf dem Startbildschirm die Bolus-Taste, geben Sie die Kohlenhydratmenge in Gramm ein und, falls erforderlich, einen Fingerstichwert. Bei Verbindung mit einem Glukosesensor wird der letzte Glukosewert automatisch eingegeben. Die Pumpe empfiehlt eine Insulindosis, die Sie jedoch überschreiben können.

Ein praktischer Tipp ist die Option der verlängerten Bolusabgabe. Bei der Bestätigung eines Bolus können Sie sich für einen verlängerten Bolus entscheiden und den Prozentsatz und die Dauer nach Bedarf anpassen.

### 3.4 Was man tun und was man nicht tun sollte

- Stellen Sie sicher, dass die maximalen Bolusgrenzen etwas über den erwarteten Bolusdosen liegen. Sie finden den maximalen Bolus in den Pumpeneinstellungen. Tippen Sie auf „Abgabegrenzen“ und dann auf „Maximaler Bolus“. Der maximale Bolus ist standardmäßig auf 10 Einheiten eingestellt, Sie können ihn jedoch zwischen 1 und 25 Einheiten einstellen.

- Es wird empfohlen, den Ziel-BG-Wert auf 110 mg/dl oder 6,5 mmol/l einzustellen, da dies mit dem Zielwert übereinstimmt, der bei Verwendung der Control-IQ-Technologie verwendet wird.

- Es wird empfohlen, dass alle Benutzer die Schnellbolusfunktion einrichten, mit der Sie einen Bolus durch einfaches Drücken der Pumpentaste abgeben können. Auf diese Weise können Sie einen Bolus durch Befolgen von Piepton-/Vibrationsbefehlen abgeben, ohne durch den Pumpenbildschirm navigieren oder diesen anzeigen zu müssen.
