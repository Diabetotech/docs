## **<u>3. Komponenten verbinden</u>**

Willkommen zu diesem Kapitel, in dem wir uns mit dem Prozess der Verbindung der verschiedenen Komponenten des Tandem Mobi-Systems befassen werden. Lassen Sie uns loslegen!

Das Tandem Mobi System funktioniert in folgenden Schritten:

- Alle 5 Minuten misst der Dexcom G6- oder G7-Sensor Ihren Blutzuckerspiegel.

- Der Sensor überträgt diese Blutzuckerwerte an die Tandem Mobi Insulinpumpe. Der Algorithmus der Pumpe berechnet die geeignete Insulindosis und passt die Insulinabgabe automatisch entsprechend an.

- Die Tandem Mobi-Pumpe kann über die Tandem Mobi-App auf Ihrem Smartphone gesteuert werden.

- Die Glukose- und Pumpendaten werden automatisch über WLAN mit der Tandem Source-Plattform synchronisiert. 

- Darüber hinaus können Sie durch die Installation der Dexcom G6- oder G7-App auf Ihrem Mobiltelefon Ihre Daten bequem an Dexcom Clarity und Dexcom Share senden. So können Pflegekräfte Ihren Blutzuckerspiegel mit der Dexcom Follow-App überwachen.

Wir führen Sie durch den Prozess der Herstellung dieser wichtigen Verbindungen. Im Einzelnen behandeln wir folgende Themen:

- So verbinden Sie die Tandem Mobi-Pumpe mit der Tandem Mobi Mobile App.

- So verbinden Sie einen Dexcom G6-Sensor mit der Tandem Mobi-Insulinpumpe.

- So verbinden Sie einen Dexcom G7-Sensor mit der Tandem Mobi-Insulinpumpe.

- So erstellen Sie ein Tandem Source-Konto.

Also, fangen wir an und lernen wir, wie man diese wichtigen Verbindungen innerhalb des Tandem Mobi-Systems herstellt.

### 3.1 So verbinden Sie die Tandem Mobi-Pumpe mit der Tandem Mobi Mobile-App

Um Ihre Mobi-Pumpe verwenden zu können, müssen Sie sie zunächst mit Ihrem iPhone koppeln. Gehen Sie dazu wie folgt vor:

- Schließen Sie Ihr Ladepad mit dem mitgelieferten USB-C-Kabel an eine Stromquelle an und legen Sie die Pumpe auf das Ladepad.

- Vergewissern Sie sich, dass Ihre Pumpe eingeschaltet ist. Sie können überprüfen, ob Ihre Pumpe eingeschaltet ist, indem Sie einmal auf die Pumpentaste drücken. Wenn die Pumpenstatusanzeigen aufleuchten, ist Ihre Pumpe eingeschaltet.

- Laden Sie die Tandem Mobi Mobile App aus dem App Store auf Ihr kompatibles iPhone herunter.

- Wenn Sie ein neuer Benutzer sind, werden Sie von der App zunächst aufgefordert, ein Konto bei Tandem Source zu erstellen.

- Tippen Sie anschließend auf „Beginnen“ und befolgen Sie die Anweisungen auf dem Bildschirm, um Ihre Mobi-Pumpe zu koppeln. 

- Zu einem bestimmten Zeitpunkt werden Sie aufgefordert, den PIN-Code Ihrer Pumpe einzugeben, den Sie unter der Seriennummer finden.

Wenn Ihre Pumpe gekoppelt ist, wird der Startbildschirm der Tandem Mobi Mobile App angezeigt.

### 3.2 So verbinden Sie einen Dexcom G6-Sensor mit der Tandem Mobi-Insulinpumpe

Bei der Tandem Mobi Insulinpumpe ist die Verbindung des Sensors mit der Pumpe ein unkomplizierter Vorgang, bei Dexcom G6 und Dexcom G7 ist dies jedoch etwas anders. So gehen Sie bei dem Dexcom G6-Sensor vor:

Verbinden Sie zunächst den Dexcom G6-Sender, indem Sie im Menü „Einstellungen“ der Pumpe auf „CGM“ und dann auf „Dexcom G6“ tippen. Tippen Sie auf das Feld „Sender-ID“ und geben Sie die eindeutige Sender-ID ein. Sie werden aufgefordert, diese zu überprüfen. Anschließend können Sie sie speichern. 

Nachdem Sie den Sender erfolgreich mit der Pumpe verbunden haben, können Sie mit dem Starten des Sensors fortfahren. So geht's:

- Wählen Sie auf dem Dashboard-Bildschirm „Einstellungen“ und dann „CGM“ aus.

- Wählen Sie „Sensor starten“ und befolgen Sie die Anweisungen.

- In diesem Schritt werden Sie aufgefordert, den Sensorkode einzugeben, der sich auf der Schutzfolie des Sensors befindet. Wenn Sie den Sensorcode verlegt haben, können Sie „Überspringen“ auswählen. In diesem Fall fordert Sie die Pumpe nach 2 Stunden auf, den Sensor zu kalibrieren, sodass Sie den Sensor weiterhin verwenden können. Die Pumpe fordert jedoch alle 24 Stunden eine Kalibrierung an. Es ist daher ratsam, den Sensorcode nicht zu verlieren, um diese zusätzlichen Schritte zu vermeiden.

- Tippen Sie auf dem Start-Sensor-Bildschirm auf „Start“. Oben in der Tandem Mobi-App wird die Meldung „Sensor-Sitzung gestartet“ angezeigt, um Sie darüber zu informieren, dass Ihre zweistündige Aufwärmphase begonnen hat. Während der Startphase des Sensors hat die Control-IQ-Technologie keinen Einfluss auf die Basalraten und gibt keine automatischen Korrekturboli ab. Der Sensor muss aktiv Messwerte liefern, damit die Control-IQ-Technologie funktionieren kann.

Wenn Sie den Sensor bereits mit der Dexcom G6 App gestartet haben, empfängt die Pumpe die Sensordaten, ohne nach einem Sensorcode zu fragen.

Wenn Sie diese Schritte befolgen, können Sie den Dexcom G6-Sender erfolgreich verbinden und den Dexcom G6-Sensor mit der Tandem Mobi-Insulinpumpe starten.

### 3.3 So verbinden Sie einen Dexcom G7-Sensor mit der Tandem Mobi-Insulinpumpe

Wenn Sie Zugang zu Dexcom G7-Sensoren haben, ist der Verbindungsvorgang noch einfacher. Wählen Sie einfach Dexcom G7 als Sensortyp aus und geben Sie den G7-Kopplungscode zweimal ein. 

Der Dexcom G7-Sensor benötigt eine 30-minütige Startphase, um sich unter Ihrer Haut anzupassen. Diese Startphase beginnt automatisch, sobald der Sensor eingesetzt wird. Wenn Sie also Ihren Dexcom G7-Sensor 30 Minuten vor der Kopplung mit Ihrer Pumpe einsetzen, erhalten Sie direkt die Sensorwerte, und die Control-IQ-Technologie muss beim Sensorwechsel nicht unterbrochen werden.

### 3.4 So erstellen Sie ein Tandem Source-Konto 

Die Pumpen- und Sensordaten können über die Tandem Source-Plattform mit einem Elternteil, Erziehungsberechtigten oder Gesundheitsdienstleister geteilt werden. 

Benutzer der Tandem Mobi Mobile App erstellen bei ihrer ersten Anmeldung in der App ein Tandem Source-Konto. Alle Sensor- und Pumpendaten werden etwa einmal pro Stunde automatisch mit der Tandem Cloud synchronisiert.

Gesundheitsdienstleister können über ihren Tandem-Vertreter ein professionelles Tandem Source-Konto beantragen. Administratoren eines Klinik-Kontos können weitere Teammitglieder hinzufügen.

Benutzer und Gesundheitsdienstleister können sich beide über sso.tandemdiabetes.com bei Tandem Source anmelden. Benutzer können eine Verbindung zu einer Klinik herstellen, indem sie zu ihren Kontoeinstellungen gehen, auf „Berichte teilen“ und dann auf „Klinik hinzufügen“ tippen. 

Obwohl der Prozess im Allgemeinen reibungslos und unkompliziert verläuft, kann es vorkommen, dass Sie auf Herausforderungen oder technische Schwierigkeiten stoßen. Denken Sie daran, dass Sie dies nicht alleine bewältigen müssen. Wenn Sie Probleme haben oder Hilfe benötigen, zögern Sie nicht, sich an den Support zu wenden. Sie können der Tandem-Facebook-Supportgruppe beitreten oder sich mit anderen Personen austauschen, die das System ebenfalls nutzen. 

Gemeinsam können Sie Erfahrungen austauschen, Rat suchen und Lösungen finden. Denken Sie daran, dass Resilienz entsteht, wenn Sie Hilfe suchen und sich ein unterstützendes Umfeld schaffen.
