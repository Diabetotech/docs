## **<u>2. Sensor anbringen</u>**

Willkommen zur ersten Lektion des Instinct-CGM-Moduls. 

In diesem Kapitel führen wir Sie durch das Einsetzen und Entfernen des Instinct-Sensors und heben die wichtigsten Informationen hervor.

### 2.1) Materialien 

Halten Sie Folgendes bereit:

- ein Instinct-Kit, das einen All-in-One-Sensorapplikator enthält; sowie 

- Alkoholtupfer.

### 2.2) Den Sensor einsetzen 

Um Ihren Sensor anzubringen, benötigen Sie die MiniMed Mobile App auf einem kompatiblen Smartphone, das mit Ihrer MiniMed 780G Pumpe gekoppelt ist.

Besuchen Sie diese Webseite, um die Gerätekompatibilität zu prüfen: medtronicdiabetes.com/app-check.

Öffnen Sie auf Ihrem Smartphone die MiniMed Mobile App. Tippen Sie auf die Menüschaltfläche und wählen Sie dann Sensor starten aus. Tippen Sie anschließend zur Bestätigung auf Ja, Instinct. Wählen Sie die Insertionsstelle an der Rückseite Ihres Oberarms. Wählen Sie einen Bereich, der flach ist, keine Narben, Muttermale oder Dehnungsstreifen aufweist und mindestens 1 Zoll von Insulininjektionsstellen entfernt ist. Bereiten Sie die Stelle vor, indem Sie sie mit normaler Seife waschen. Lassen Sie sie trocknen. Reinigen Sie die Stelle dann mit einem Alkoholtupfer und lassen Sie sie an der Luft trocknen. Schrauben Sie anschließend die Kappe vom Sensor-Applikator ab. Setzen Sie den Applikator über die Stelle und drücken Sie ihn dann fest nach unten, um den Sensor anzubringen. Ziehen Sie den Applikator vorsichtig ab und vergewissern Sie sich, dass der Sensor sicher sitzt.

### 2.3) Entfernen des Sensors 

Um den Instinct-Sensor zu entfernen, ziehen Sie den Rand des Klebestreifens hoch, der den Sensor auf Ihrer Haut hält. Ziehen Sie ihn langsam in einer Bewegung von Ihrer Haut ab. Entsorgen Sie den verwendeten Sensor gemäß den Anweisungen Ihres Arztes oder Ihrer Ärztin.

### 2.4) Was man tun und was man nicht tun sollte 

- Um das Risiko ausbleibender Ergebnisse und/oder einer Infektion zu vermeiden, verwenden Sie das Sensor-Kit-Paket oder den Sensor-Applikator nicht, wenn: 

- es beschädigt zu sein scheint, 

- das Manipulationssiegel darauf hinweist, dass der Sensor-Applikator bereits geöffnet wurde, oder 

- der Inhalt des Sensor-Kits das Verfallsdatum überschritten hat. 

- Für die beste Verbindung ist es wichtig, dass Sie Pumpe und Sensor auf derselben Körperseite tragen. Ihr Infusionsset können Sie auf beiden Seiten tragen. 

- Drücken Sie den Sensor-Applikator erst herunter, wenn er über der vorbereiteten Stelle positioniert ist, um unbeabsichtigte Ergebnisse oder Verletzungen zu vermeiden.

- Der Sensor ist so konzipiert, dass er bestimmte Zustände erkennt, die auftreten können, wenn der Sensor nicht wie vorgesehen funktioniert und sich abschalten muss. Dies kann passieren, wenn sich der Sensor von der Haut löst, oder wenn der Sensor erkennt, dass er möglicherweise nicht wie vorgesehen arbeitet. Wenn Sie über den Zustand informiert werden, ersetzen Sie Ihren Sensor. Wenden Sie sich an den Customer Service von MiniMed, wenn Sie vor Ablauf der Tragedauer eine Meldung „Sensor ersetzen“ erhalten. 

- Intensive körperliche Aktivität kann dazu führen, dass sich Ihr Sensor durch Schweiß oder Sensorbewegung lockert. Wenn sich der Sensor lockert oder wenn die Sensorspritze aus Ihrer Haut herauskommt, erhalten Sie möglicherweise keine Messwerte oder unzuverlässige Messwerte, die nicht dazu passen, wie Sie sich fühlen. Prüfen Sie, ob sich Ihr Sensor nicht gelöst hat. Wenn er sich gelöst hat, entfernen Sie ihn, setzen Sie einen neuen ein und wenden Sie sich an den Customer Service. 

- Nach dem Entfernen des Sensors können verbleibende Klebereste auf der Haut mit warmem Seifenwasser oder Isopropylalkohol entfernt werden.

Sie kennen jetzt das richtige Vorgehen zum Einsetzen und Entfernen Ihres Instinct-Sensors. Wenn Sie diese bewährten Vorgehensweisen befolgen, erhalten Sie jeden Tag zuverlässige Glukosewerte und können so sichere, fundierte Entscheidungen über Ihre Behandlung treffen.
