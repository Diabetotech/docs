## **<u>5. Datenaustausch</u>**

Willkommen zu dieser Lektion über die gemeinsame Nutzung von Daten mit dem Simplera-Sensor.

In diesem Video erfahren Sie, wie Sie Daten mit Ihrem medizinischen Betreuer austauschen, Berichte erstellen und Daten mit Familienmitgliedern oder Pflegepersonal austauschen können.

### 5.1 Austausch von Daten mit Ihrem medizinischen Betreuer

Um Ihre Daten mit Ihrem medizinischen Betreuer auszutauschen, bietet Medtronic die CareLink Plattform an. Als Nutzer oder Familienmitglied können Sie ein CareLink Personal-Konto unter carelink.minimed.com erstellen. Als Gesundheitsdienstleister können Sie ein CareLink-Klinik-System-Konto unter carelink.medtronic.eu erstellen, oder über carelink.medtronic.com, wenn Sie in den USA leben.

Wenn Sie im Gesundheitswesen tätig sind, können Sie Ihre Patienten zum CareLink-Klinik-System hinzufügen, indem Sie deren Namen und Geburtsdatum eingeben.

- Sie können dann den Patienten einladen, seine Daten zu teilen, indem Sie eine E-Mail über CareLink senden.

- Der Patient kann die Einladung annehmen und den Code aus der E-Mail in seinen persönlichen CareLink-Kontoeinstellungen oder über die MiniMed Mobile App eingeben, insbesondere im Abschnitt „Daten mit Krankenhäusern teilen“.

Wenn Sie die Simplera-App oder die MiniMed Mobile App verwenden, werden diese bei der Ersteinrichtung mit Ihrem persönlichen CareLink-Konto verknüpft. Wenn Ihr persönliches CareLink-Konto mit dem Konto Ihres Krankenhauses verbunden ist, kann Ihr medizinischer Betreuer Ihre Daten in Echtzeit einsehen.

### 5.2 Erstellen von Berichten

Sobald Ihre Daten in CareLink hochgeladen sind, können Sie Berichte erstellen, indem Sie die Option auswählen. Das Erstellen und Interpretieren von Berichten wird im Video Berichte des automatischen Insulinverabreichungssystems MiniMed 780G ausführlich erklärt.

### 5.3 Weitergabe von Daten an Angehörige

Um Simplera-Daten mit Familienmitgliedern oder Pflegekräften zu teilen, haben Sie zwei Möglichkeiten.

- Erstens können Sie CareLink-Daten auf Ihrem PC einsehen, indem Sie als Pflegepartner ein CareLink Personal-Konto anlegen. Sie müssen den Zugriff auf die CareLink-Daten des Nutzers, den Sie verfolgen möchten, beantragen, indem Sie dessen Benutzernamen eingeben. Der Nutzer kann Sie dann über seine eigene CareLink-Plattform als Betreuer genehmigen.

- Wenn Sie die Simplera App oder die MiniMed Mobile App verwenden, können Sie Ihre Daten auch mit bis zu 5 Pflegepartnern über die CareLink Connect App teilen. Zu Beginn müssen die Pflegepartner die CareLink Connect App auf ihr kompatibles Mobiltelefon herunterladen. Dann müssen sie ein CareLink-Konto erstellen und eine Folgeanfrage senden, indem sie den Benutzernamen des Patienten eingeben, dem sie folgen möchten. Der Patient muss diese Anfrage über seine MiniMed Mobile- oder Simplera-App oder direkt über sein CareLink-Konto auf einem Computer annehmen.

Die Nutzung dieser Sharing-Funktionen verbessert die Zusammenarbeit, die Widerstandsfähigkeit und die fundierte Entscheidungsfindung für eine bessere Gesundheit.
