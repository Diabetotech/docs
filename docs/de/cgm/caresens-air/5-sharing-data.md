## **<u>5. Daten teilen</u>**

Willkommen zu dieser Lektion über die Weitergabe von Daten mit dem CareSens Air CGM. In diesem Abschnitt wird erklärt, wie Sie Ihre Blutzuckerdaten über die Sens365-App und Sens365 Web an Ihren Arzt und Ihre Angehörigen weitergeben können. Außerdem gehen wir auf die Details der verfügbaren Berichte ein. 

### 5.1 Weitergabe von Daten an Ihren Gesundheitsdienstleister über die Sens365-App

Pflegekräfte können die Sens365-App auf ihren Smart-Geräten installieren, um Ihre Glukosedaten zu empfangen und zu überwachen. Dazu müssen Sie zunächst einen sechsstelligen Code in der CareSens Air-App generieren.

Tippen Sie unten auf dem Startbildschirm auf „Einstellungen“. Tippen Sie auf „Pflegeanbieter“. Tippen Sie auf „Verbinden“, um Ihr Konto mit dem Konto Ihres Pflegeanbieters zu verknüpfen. Tippen Sie auf „Freigabecode generieren“. Wählen Sie die Daten aus, die Sie mit Ihrem Pflegeanbieter teilen möchten, und tippen Sie dann auf „Freigabecode generieren“. Teilen Sie anschließend den auf dem Bildschirm angezeigten Code mit Ihrem Pflegeanbieter. 

Geben Sie nun den Freigabecode in die Sens365-App ein. Nachdem Sie die Sens365-App auf dem Smart-Gerät des Pflegepersonals gestartet haben, tippen Sie auf das Personensymbol unten auf dem Startbildschirm. Tippen Sie auf „Hinzufügen”. Geben Sie den vom Sensorbenutzer bereitgestellten Freigabecode ein und tippen Sie auf „OK”. Sie können den Freigabecode auch über die Einstellungen unten auf dem Startbildschirm eingeben. Tippen Sie auf „Verbindungen”. Tippen Sie auf das Symbol „Person hinzufügen” in der oberen rechten Ecke des Bildschirms. Geben Sie den vom Sensorbenutzer bereitgestellten Freigabecode ein. Tippen Sie auf “OK”, um den Vorgang abzuschließen. Nach Abschluss des Vorgangs wird eine Benachrichtigung in Ihrer CareSens Air-App angezeigt.

Sie können auch den von Ihrem Gesundheitsdienstleister bereitgestellten Code in Ihrer CareSens Air-App eingeben, um eine Verbindung zu ihm herzustellen. Öffnen Sie die CareSens Air-App, tippen Sie in den Einstellungen auf „Gesundheitsdienstleister“ und dann auf „Verbinden“. Tippen Sie auf „Freigabecode eingeben“. Geben Sie den Code Ihres Gesundheitsdienstleisters ein. Wenn der Code gültig ist, wird auf dem Bildschirm eine Bestätigungsmeldung des Gesundheitsdienstleisters angezeigt. Vergewissern Sie sich, dass Sie den richtigen Gesundheitsdienstleister hinzufügen. Nach Abschluss des Vorgangs wird eine Abschlussbenachrichtigung angezeigt. 

Sie können die Weitergabe Ihrer Glukosewerte an Ihren Gesundheitsdienstleister vorübergehend unterbrechen oder die weitergegebenen Daten ändern. Tippen Sie unten auf dem Startbildschirm auf „Einstellungen“ und anschließend auf „Gesundheitsdienstleister“. Tippen Sie auf dem Bildschirm „Gesundheitsdienstleister“ auf den Gesundheitsdienstleister, dessen Freigabeoptionen Sie ändern möchten.

Tippen Sie auf „Daten teilen“, um die Datenfreigabe zu aktivieren oder zu deaktivieren. Tippen Sie auf die Kontrollkästchen, um die Freigabe von Daten zu Blutzuckerwerten und Ereignissen oder Warnmeldungen zu beenden. 

Wenn Sie keine Daten mehr mit einem Gesundheitsdienstleister teilen möchten, können Sie die Freigabe deaktivieren. Tippen Sie auf dem Bildschirm „Gesundheitsdienstleister“ auf das Symbol „Löschen“ neben dem Dienstleister, mit dem Sie keine Daten mehr teilen möchten. Tippen Sie im Popup-Fenster auf „Trennen“, um den Vorgang abzuschließen. 

### 5.2 Weitergabe von Daten an Ihren Gesundheitsdienstleister über das Sens365-Webportal

Alle Daten, die von der CareSens Air-App auf Ihrem Smart-Gerät gespeichert werden, werden automatisch auf dem Cloud-Server gespeichert. Um den Upload Ihrer App-Daten zu aktualisieren, tippen Sie unten auf dem Startbildschirm auf „Einstellungen“ und anschließend auf „Hochladen“.

Sie können Ihre hochgeladenen Glukosedaten über die cloudbasierte Plattform Sens365 Web anzeigen und verwalten. Öffnen Sie Ihren Webbrowser und rufen Sie https://sens365.i-sens.com auf. Klicken Sie auf dem Startbildschirm auf „Erste Schritte“.

Wenn Sie die Website zum ersten Mal besuchen, werden Sie aufgefordert, Ihr .Wohnsitzland und Ihre Sprache auszuwählen. Klicken Sie auf „Senden“, um Ihre Auswahl zu bestätigen. 

Klicken Sie auf dem Anmeldebildschirm auf „Konto erstellen“, wenn Sie noch kein Sens365-Konto haben. Geben Sie Ihre E-Mail-Adresse ein und bestätigen Sie sie. Akzeptieren Sie die erforderlichen Bedingungen, erstellen Sie Ihr Passwort und geben Sie alle erforderlichen Informationen ein. Es erscheint ein Pop-up-Fenster mit der Meldung „Registrierung erfolgreich abgeschlossen“.

Wenn Sie bereits ein Sens365-Konto haben, geben Sie einfach Ihre registrierte E-Mail-Adresse und Ihr Passwort auf dem Bildschirm ein, um sich anzumelden. 

Bei Ihrer ersten Anmeldung bei Sens365 Web wird der Bildschirm zur Auswahl des Benutzermodus angezeigt. Wenn Sie CareSens Air-Benutzer sind, wählen Sie „Sensorbenutzer”. Wenn Sie Pflegekraft sind, wählen Sie „Pflegekraft”.

Sensorbenutzer müssen auf „OK“ klicken, um ihr Wohnsitzland und die Glukosemesseinheit zu bestätigen. Anschließend werden sie zum Startbildschirm weitergeleitet.

Um eine Verbindung zu einem neuen Pflegedienstleister herzustellen, klicken Sie in der Navigationsleiste links auf „Verbindungen“. Klicken Sie rechts auf die Schaltfläche „Mit Pflegedienstleister verbinden“. Geben Sie den Freigabecode des Pflegedienstleisters ein und klicken Sie auf „OK“. Überprüfen Sie den Namen der Freigabegruppe. Nachdem Sie die Vereinbarung gelesen haben, aktivieren Sie das Kontrollkästchen und klicken Sie auf „OK“. Nach Abschluss des Vorgangs finden Sie den neuen Pflegedienstleister in der Liste.

Um die Verbindung zu einem Pflegedienstleister zu trennen, klicken Sie auf die Schaltfläche „Trennen“ und anschließend im Popup-Fenster auf „Trennen“.

Lassen Sie uns nun den Modus „Care Provider“ (Pflegeanbieter) besprechen. Wenn Sie sich anmelden, werden auf dem Bildschirm Gruppennamen und eine Liste der Sensorbenutzer in jeder Gruppe angezeigt. Außerdem wird eine Zusammenfassung der Informationen zu jedem Benutzer angezeigt. Sie können eine Spalte per Drag & Drop verschieben, um sie neu zu positionieren.

Wenn Sie auf die Schaltfläche „Dashboard“ für einen bestimmten Benutzer klicken, wird das Popup-Fenster „Zum Dashboard hinzufügen“ angezeigt. Weisen Sie dem Benutzer einen Pflegedienst zu und klicken Sie dann auf „OK“. Das Dashboard wird auf der rechten Seite des Bildschirms angezeigt. Sie können das Dashboard jederzeit öffnen, indem Sie auf die Schaltfläche „Dashboard“ in der oberen rechten Ecke klicken.

Gruppen können erstellt oder gelöscht werden. Um eine Gruppe zu erstellen, klicken Sie auf die drei Punkte neben dem Gruppennamen. Passen Sie im Popup-Fenster „Gruppeneinstellungen“ den Gruppennamen an, generieren Sie den Freigabecode der Gruppe und klicken Sie dann auf „Speichern“.

Sie können auch einen neuen Sensorbenutzer hinzufügen oder einen zur Gruppe einladen. Wenn Sie einen Sensorbenutzer hinzufügen, geben Sie den Freigabecode ein, den Sie von ihm erhalten haben. Wenn Sie einen Sensorbenutzer einladen, bestätigen Sie den Gruppennamen und senden Sie dann Ihren Freigabecode an den Benutzer. Sie können den Code kopieren oder direkt per E-Mail einladen. Der Freigabecode Ihres Gesundheitsdienstleisters bleibt unverändert, sofern Sie ihn nicht ändern. Bei Bedarf können Sie einen neuen Freigabecode generieren.

Darüber hinaus können Sie einen weiteren Pflegedienstleister einladen, die Benutzergruppen gemeinsam mit Ihnen zu verwalten. Klicken Sie dazu auf das Drei-Punkte-Menü neben dem Gruppennamen und wählen Sie „Pflegedienstleister einladen”. Geben Sie die E-Mail-Adresse des Pflegedienstleisters ein und wählen Sie dann „Admin” oder „Koordinator”. Aktivieren Sie das Kontrollkästchen „Bestätigung” und klicken Sie auf „Senden”. 

### 5.3 Berichte

Klicken Sie im Modus "Care Provider" auf den Namen des Sensorbenutzers, um den Bildschirm Reports aufzurufen. Standardmäßig gelangen Sie zum Bericht AGP. Auf der linken Seite finden Sie „Übersicht“. Auf der rechten Seite können Sie auf „Vergleich“, „Muster“, „Trends“, „Statistiken“ und „Ereignisse“ zugreifen.

In der Zeile darüber können Sie den Zeitraum einstellen, den Sie anzeigen möchten.

Die Seite “Übersicht” zeigt Zeit im Zielbereich, durchschnittlichenr Glukosewert, GMI, Glukosevariabilität und Standardabweichung an. Täglicher Durchschnittsglukosewert, Insulin, tägliche Zeit im Bereich, Zeit mit aktivem CGM und Geräteinformationen werden ebenfalls angezeigt. Es gibt eine Anpassungsoption, mit der Sie das Layout durch Hinzufügen, Löschen oder Verschieben von Blöcken anpassen können.

AGP ist ein standardisierter Glukosebericht, der Zeit im Zielbereich, Zeit mit aktivem CGM, Glukosewerte, ambulantes Glukoseprofil und tägliche Glukoseprofile zeigt. Sie können den Prozentsatz der Zeit, die innerhalb des Zielbereichs für den Glukosespiegel verbracht wurde, anhand des TIR-Diagramms überprüfen und den A1c-Wert mithilfe von GMI schätzen.

Auf der Seite "Vergleich"  werden zwei Diagramme basierend auf den ausgewählten Zeiträumen angezeigt. So lassen sich Veränderungen im Glukosetrend und Verbesserungen im Glukosemanagement durch den Vergleich der beiden Diagramme leicht erkennen.

Auf der Seite "Muster" werden die niedrigen, Ziel- und hohen Glukosewerte für den ausgewählten Zeitraum im Diagramm markiert. So können Sie Muster erkennen, indem Sie sehen, wann niedrige oder hohe Glukosewerte auftreten und welche Ereignisse mit Glukoseveränderungen in Verbindung stehen könnten.

Die Seite „Trends“ zeigt tägliche und wöchentliche Glukosetrends an. Sie können die täglichen Schwankungen anzeigen oder alle täglichen Trends zusammen in einem Wochen-Diagramm anzeigen.

Die Seite "Statistiken" zeigt die Glukosewerte nach Wochentag und Tageszeit an. Sie können die täglichen und stündlichen durchschnittlichen Glukosewerte innerhalb des ausgewählten Zeitraums einsehen.

Die Seite “Ereignisse” zeigt alle in der App CareSens Air aufgezeichneten Ereignisprotokolle in chronologischer Reihenfolge an.

Weiter oben auf dem Bildschirm, neben dem Namen des Sensorbenutzers, klicken Sie auf „Zielbereichseinstellungen“, um den Zielblutzuckerbereich für den Tag und die Nacht anzupassen und die Verwaltung auf die Ziele des Benutzers zuzuschneiden.

Sowohl im Sensorbenutzer- als auch im Pflegeanbieter-Modus können Sie Berichte herunterladen, ausdrucken und versenden sowie Daten exportieren. Um einen Bericht herunterzuladen, klicken Sie auf das Abwärtspfeil-Symbol in der oberen rechten Ecke; der Bericht wird als PDF gespeichert. Zum Ausdrucken klicken Sie auf das Druckersymbol. Um den Bericht zu versenden, klicken Sie auf das Briefumschlag-Symbol; er wird als PDF an die von Ihnen angegebene E-Mail-Adresse gesendet. Um Ihre Daten im CSV-Format zu exportieren, klicken Sie schließlich auf das Symbol „Daten exportieren“.

Auf dem CareSens Air Receiver können Sie über die Option „Statistik“ im Hauptmenü Berichte für die letzten 7, 14, 30 oder 90 Tage anzeigen. Jeder Bericht zeigt Ihren durchschnittlichen Blutzuckerwert, die Standardabweichung (SD) und den Variationskoeffizienten (CV) für den ausgewählten Zeitraum sowie Ihre Zeit im Zielbereich an.

### 5.4 Tipps

Lesen Sie diese Hinweise, bevor Sie Ihre CareSens Air-Daten weitergeben:

- Wenn Sie über die CareSens-App eine Verbindung zu einem Gesundheitsdienstleister herstellen, beträgt die Frist für die Eingabe des generierten Freigabecodes 5 Minuten. Wenn der Code abläuft, erstellen Sie einen neuen.

- Wenn Sie Ihre Freigabeoptionen für einen Pflegedienstleister ändern, wird dieser über die Änderungen informiert.

- Sens365 ist ein sekundärer Viewer für das CareSens Air CGM-System und dient nicht dazu, Behandlungsentscheidungen zu treffen oder professionelle medizinische Beratung zu ersetzen. Sens365 garantiert nicht den Echtzeitempfang der Glukosedaten des CareSens Air CGM-Benutzers.

- Sie können Daten nicht direkt vom CareSens Air Receiver aus teilen. Um auf die auf dem Gerät gespeicherten Daten zuzugreifen, besuchen Sie die Sens365-Website, melden Sie sich an und laden Sie die Daten vom Empfänger auf den Cloud-Server hoch.

In dieser Lektion haben wir verschiedene Optionen zum Teilen und Hochladen Ihrer CareSens Air-Daten vorgestellt. Mit diesen Funktionen können Sie enger mit Ihrem Betreuungsteam zusammenarbeiten, was Ihnen hilft, zeitnahe und fundierte Entscheidungen für eine bessere Gesundheit zu treffen.
