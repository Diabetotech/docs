## **<u>3. Sensor starten</u>**

In dieser Lektion zeigen wir Ihnen, wie Sie den TouchCare Nano-Sensor in Betrieb nehmen.

Sie können den Sensor entweder über den PDM oder die EasyPatch-App anschließen. Wir erklären Ihnen alles Schritt für Schritt.

Wenn Sie keine TouchCare Nano-Pumpe haben, können Sie auch die EasySense-App verwenden. Da die meisten Menschen jedoch den Sensor mit der Pumpe verwenden, wird in dieser Anleitung nicht auf die EasySense-App eingegangen.

### 3.1 PDM

Der PDM (Personal Diabetes Manager) ist ein kleines Gerät, das sowohl an den TouchCare Nano Sensor als auch an die Pumpe angeschlossen werden kann.

Er verfügt über einen Touchscreen und eine Home-Taste auf der Vorderseite, eine Einschalttaste auf der Oberseite und einen Ladeanschluss und eine Schallöffnung auf der Unterseite. In der oberen linken Ecke befindet sich außerdem eine Kontrollleuchte, die in verschiedenen Situationen grün, gelb oder blau blinken kann. Beim Laden blinkt sie beispielsweise blau, während ein grünes Dauerlicht volle Ladung anzeigt.

- Um das Gerät auszuschalten, drücken Sie den Netzschalter etwa zwei Sekunden lang und schieben ihn anschließend zum Ausschalten. Alternativ halten Sie die Einschalttaste etwa acht Sekunden gedrückt.

- Wenn der Bildschirm schwarz wird, drücken Sie die Einschalttaste und schieben Sie dann zum Entsperren.

Gehen Sie folgendermaßen vor, um den TouchCare Nano Sensor mit dem PDM zu starten:

- Schließen Sie zunächst Ihren Sender an den PDM an. Streichen Sie auf dem Hauptbildschirm nach links. Tippen Sie im Hauptmenü auf Sensor. Tippen Sie auf Transmitter SN. Geben Sie manuell die Seriennummer ein, die auf dem Sender oder auf dem Verpackungsetikett des Senders aufgedruckt ist.

- Verbinden Sie als nächstes Ihren Sensor mit dem PDM. Streichen Sie auf dem Hauptbildschirm nach links. Tippen Sie im Hauptmenü auf Sensor.

- Tippen Sie auf Sensor verbinden. Tippen Sie auf Eingabe. Geben Sie den Sensorcode aus der Sensorverpackung ein. Tippen Sie auf Fertig, um den Sensor zu verbinden. Tippen Sie auf Weiter.

- Legen Sie als Nächstes den TouchCare Nano-Sensor und den Sender auf Ihre Haut, falls Sie dies noch nicht getan haben.

- Nach der Aufwärmphase, die bis zu 30 Minuten dauert, erhalten Sie Glukosemesswerte. Auf Ihrem Hauptbildschirm sehen Sie:

- Ihren aktuellen Blutzuckerspiegel zusammen mit einem Trendpfeil. Wenn Ihr Messwert unter 40 mg/dL (oder 2,2 mmol/L) liegt, wird "NIEDRIG" angezeigt, und wenn er über 400 mg/dL (oder 22,2 mmol/L) liegt, wird "HOCH" angezeigt. Überprüfen Sie diese extremen Messwerte immer mit einem Fingerstich und handeln Sie entsprechend.

- Auf dem Startbildschirm des PDMs wird außerdem eine Schätzung des Insulins angezeigt, das Sie noch an Bord haben, sowie eine Glukose-Trendkurve, die unter Ihrem aktuellen Messwert angezeigt wird.

- Sie können auch:

- Drücken Sie lange auf das Trenddiagramm, um eine größere Ansicht im Querformat zu öffnen.

- Wischen Sie nach links, um das Hauptmenü zu öffnen.

- Oder wischen Sie nach rechts, um den Statusbildschirm anzuzeigen.

- Bevor Sie einen Sensor von Ihrem Körper entfernen, vergewissern Sie sich, dass er abgekoppelt wurde.  Gehen Sie zum Hauptmenü Ihres PDMs, wählen Sie dort „Sensor“ aus und anschließend die Option „Sensor trennen“. Falls die Option „Sensor trennen“ nicht angezeigt wird, überspringen Sie diesen Schritt.

### 3.2 EasyPatch-App

Die EasyPatch-App kann sowohl mit dem TouchCare Nano-Sensor als auch mit der Pumpe verbunden werden. Sie können sie auf die meisten iPhones und Android-Telefone herunterladen. Um zu überprüfen, ob Ihr Telefon mit der App funktioniert, besuchen Sie medtrum.com/kompatibilität. Sie können die App finden, indem Sie im App Store oder bei Google Play nach "Medtrum" suchen.

Gehen Sie wie folgt vor, um den TouchCare Nano-Sensor über die App zu starten:

- Öffnen Sie die Medtrum EasyPatch App, und registrieren Sie sich oder melden Sie sich bei Ihrem Konto an.

- Verbinden Sie zunächst den Sender mit der EasyPatch-App. Gehen Sie auf dem Startbildschirm zum Hauptmenü. Tippen Sie auf Einstellungen und dann auf CGM-System. Vergewissern Sie sich zunächst, dass die Werkskalibrierung aktiviert ist, damit Sie die Blutzuckermesswerte des Sensors sofort nach dem Aufwärmen und ohne manuelle Kalibrierung erhalten können. Tippen Sie auf „Sender SN hinzufügen“. Sie können den QR-Code der Seriennummer, der sich auf dem Etikett der Senderverpackung befindet, scannen. Tippen Sie anschließend auf „OK“. Sie können auch die Seriennummer, die auf dem Sender oder auf dem Verpackungsetikett des Senders aufgedruckt ist, manuell eingeben.

- Als nächstes verbinden Sie Ihren Sensor mit der EasyPatch-App. Tippen Sie auf dem Startbildschirm auf Hauptmenü, auf CGM, auf Sensor verbinden und auf Code eingeben. Sie können den QR-Code des Sensors auf der Sensorverpackung scannen oder den Sensorcode manuell eingeben. Wenn Sie den Code scannen, tippen Sie auf OK, um zu bestätigen, dass der Code korrekt ist, und die App kehrt zum Startbildschirm zurück. Wenn Sie den Code manuell eingeben, tippen Sie auf Fertig, um zum Startbildschirm zurückzukehren.

- Legen Sie als Nächstes Ihren TouchCare Nano-Sensor  und den Sender auf Ihre Haut, falls Sie dies noch nicht getan haben.

- Nach der Aufwärmphase, die bis zu 30 Minuten dauert, erhalten Sie Glukosemesswerte. Auf Ihrem Startbildschirm wird angezeigt:

- Der aktuelle Blutzuckermesswert mit einem Trendpfeil. Messwerte unter 40 mg/dl oder 2,2 mmol/L werden als "NIEDRIG" angezeigt, Messwerte über 400 mg/dl oder 22,2 mmol/L als "HOCH". Überprüfen Sie extreme Messwerte immer mit einem Fingerabdruck und ergreifen Sie die erforderlichen Maßnahmen.

- Auf dem Startbildschirm wird unter dem aktuellen Messwert auch eine Glukose-Trendkurve angezeigt.

- Im Querformat können Sie die Glukosetrends der letzten 24 Stunden anzeigen.

- Bevor Sie einen Sensor von Ihrem Körper entfernen, vergewissern Sie sich, dass er in Ihrer Medtrum-App getrennt wurde. Gehen Sie dazu zum Hauptmenü. Wählen Sie den Punkt CGM aus und dann die Option „Sensor trennen“. Falls die Option „Sensor trennen“ nicht angezeigt wird, können Sie diesen Schritt überspringen.

### 3.3 Tipps und Tricks

- Der Sensor ist bis zu einer Tiefe von 2,5 Metern für 60 Minuten wasserdicht, so dass Sie duschen und schwimmen können.

- Wenn Sie den PDM verwenden, sollten Sie ihn mit dem mitgelieferten Ladegerät aufladen. Das erste Aufladen dauert in der Regel etwa 2 Stunden und kann das Gerät bis zu einer Woche mit Strom versorgen. Der PDM kann Daten von bis zu 90 Tagen speichern. Selbst wenn die Batterie leer ist oder ein Systemfehler auftritt, bleiben Ihre Daten sicher. Damit alles reibungslos funktioniert, stellen Sie sicher, dass Datum und Uhrzeit richtig eingestellt sind. Sie können das Datum und die Uhrzeit über das Menü "Einstellungen" einstellen, wenn kein Patch aktiv ist.

- Wenn Sie die EasyPatch-App verwenden, werden Sie möglicherweise feststellen, dass sich der Akku Ihres Telefons etwas schneller entlädt. Auch wenn sich Ihr Telefon ausschaltet, bleiben Ihre Daten sicher und werden aktualisiert, sobald die App neu gestartet wird. Die App kann bis zu 90 Tage lang Daten speichern, sodass Sie keine wertvollen Informationen verlieren.

- Damit die EasyPatch-App eine Verbindung zu Ihrem Sensor herstellen kann, ist Bluetooth-Kommunikation erforderlich. Die App kann eine Verbindungsunterbrechung erkennen und Sie darüber informieren. Sobald das Problem behoben ist, wird die Kommunikation wieder aufgenommen.

- Der Sensor kann bei Bedarf kalibriert werden.

- Auf dem PDM: Gehen Sie zum Hauptmenü, tippen Sie auf Sensor und wählen Sie dann Sensorkalibrierung. Hier können Sie Ihren Blutzuckerwert aus dem Fingerstich eingeben, auf Fertig tippen und auf Ja tippen, um Ihren Sensor zu kalibrieren.

- In der EasyPatch-App: Tippen Sie auf die Verknüpfungstaste BZ eingeben, die sich im unteren Menü des Startbildschirms befindet. Hier können Sie Ihren Blutzuckerwert aus dem Fingerstich eingeben, auf Fertig tippen und auf OK tippen, um Ihren Sensor zu kalibrieren.

- In der App können Notizen gemacht werden. Tippen Sie im Hauptmenü auf Ereignisse, um Ereignisse wie Nahrungsaufnahme, Bewegung oder Insulinabgabe für umfassende Daten hinzuzufügen.

Wenn Sie diese Schritte befolgen und bei Problemen proaktiv vorgehen, können Sie Ihren TouchCare Nano-Sensor optimal nutzen und eine effektive, kontinuierliche Blutzuckermessung durchführen.
