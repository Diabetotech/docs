## **<u>5. Daten teilen</u>**

Willkommen zur Lektion über die Datenfreigabe mit dem Sibionics GS3 CGM. Wir zeigen Ihnen, wie Sie Ihre Sensordaten für Ihren Arzt und Ihre Angehörigen freigeben. Außerdem erklären wir, wie Sie Berichte mit dem GS3-System erstellen. Los geht’s.

### 5.1 Weitergabe von Daten an Ihren Gesundheitsdienstleister 

Tippen Sie auf das Profil-Symbol auf dem Startbildschirm der App. Tippen Sie unter „Verbindungen“ auf „Gesundheitsversorgung“, um einen Gesundheitsdienstleister einzuladen, Ihre Sensor-Blutzuckerwerte und Trendgrafiken anzuzeigen. Tippen Sie auf „Mit neuer Klinik verbinden“ und geben Sie die Praxis-ID Ihrer Klinik ein.

Als Gesundheitsdienstleister können Sie über proview.sisensing.com/login ein kostenloses Konto bei SIBIONICS Proview erstellen. Sie können sich als Gesundheitsdienstleister registrieren, um auf die SIBIONICS ProView-Plattform zuzugreifen, und über das Menü Ihre Praxis anlegen. Wenn Sie eine Praxis anlegen, können Sie Ihre Praxis-ID festlegen, die Sie später Ihren Patienten mitteilen können. 

Wenn Sie auf das SIBIONICS ProView-Logo tippen, gelangen Sie zur Startseite, auf der Sie Ihre Patienten sehen können. Über die Registerkarte „Patienten“ können Sie neue Patienten hinzufügen. In Ihrer Patientenliste können Sie die Ansicht so einstellen, dass nur mit Sternchen markierte Patienten oder Patienten mit einem bestimmten Glukosemuster angezeigt werden. Auf der Startseite von SIBIONICS ProView können Sie auf einen bestimmten Patienten tippen, um dessen Daten anzuzeigen.

### 5.2 Weitergabe von Daten an Ihre Familie

### Zum Zeitpunkt der Erstellung dieses Kurses ist die Echtzeit-Freigabe von GS3-Daten für Familienmitglieder noch nicht verfügbar, wird aber in Kürze veröffentlicht. Sobald diese Funktion aktiviert ist, können Sie über die Registerkarte „Health Care Connections“ einen SIBIONICS Companion auswählen und einen Follower einladen, der Ihre Daten aus der Ferne überwachen kann.

### 5.3 Berichte erstellen

Die Tagesdaten zeigen den Durchschnitt Ihrer Glukosewerte aus dem Sensor sowie Ihre Zeit im Zielbereich. Unter „Tägliche Trends“ sehen Sie die täglichen Glukosewerte als Grafik. Dort werden auch Symbole für Ereignisse an den jeweiligen Zeitpunkten angezeigt, zu denen sie aufgezeichnet wurden.

Um das Datum für einen Bericht auszuwählen, tippen Sie oben auf dem Bildschirm auf „Zurück“ oder „Weiter“, um durch die täglichen Daten zu blättern, oder tippen Sie auf die Datumsschaltfläche, um einen bestimmten Tag zur Überprüfung auszuwählen.

AGP-Berichte fassen Daten über mehrere Tage zusammen und zeigen das Muster und die Schwankungen der Sensor-Glukosewerte an einem typischen Tag. Der grüne Bereich im AGP stellt den Zielbereich für den Glukosespiegel dar. Beachten Sie, dass das AGP mindestens 7 Tage lang Glukosemessungen benötigt, um statistische Validität zu gewährleisten.

Um einen AGP-Bericht zu exportieren, tippen Sie in der GS3-App auf „Bericht“. Legen Sie den Zeitbereich auf 7, 14, 30 oder 90 Tage fest oder tippen Sie auf „Benutzerdefiniert“, um das Start- und Enddatum manuell anzupassen. Tippen Sie anschließend auf „Bericht anzeigen“, um den Bericht für den ausgewählten Zeitraum zu erstellen. Sie können den Bericht über die Symbole in der oberen rechten Ecke des Bildschirms herunterladen oder teilen.

In der SIBIONICS ProView-Plattform können Sie drei Arten von Berichten anzeigen, wenn Sie auf der Startseite auf einen Patienten klicken: AGP-Bericht, Vergleichsbericht und täglicher Glukosewertbericht. Für jeden Bericht kann der Datumsbereich geändert werden. Im Vergleichsbericht können Sie die AGP-Details mit einem früheren Zeitraum vergleichen. Im Tagesbericht können Sie mit der Maus über die Glukosekurve fahren, um den spezifischen Glukosewert sowie spezifische Aufzeichnungsinformationen wie Bilder der Mahlzeiten anzuzeigen, sofern diese zu diesem Zeitpunkt aufgenommen wurden. Schließlich können die Berichte jederzeit über das Download-Symbol in der oberen rechten Ecke heruntergeladen werden.

In dieser Lektion haben wir die Funktionen des Sibionics GS3 CGM zum Teilen von Glukosedaten und Erstellen von Berichten vorgestellt. Durch die Nutzung dieser Funktionen können Sie und Ihr Betreuungsteam effektiver zusammenarbeiten und zeitnah datengestützte Entscheidungen treffen, die zu einer besseren Gesundheit beitragen.
