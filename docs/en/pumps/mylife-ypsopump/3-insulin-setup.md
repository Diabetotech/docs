**<u>3. Insulin setup (5 min)</u>**

In this video, we're exploring how to program insulin delivery on the mylife YpsoPump.

Do bear in mind: If you plan to use this pump as a part of the mylife Loop automated insulin delivery system, insulin delivery settings will be managed via the mylife CamAPS FX app. A dedicated module covers this in detail. However, in case you forget your phone, understanding how to manually control the pump is critical, and that's our focus in this video.

We'll cover:

- Programming basal insulin,

- Starting and stopping a temporary basal,

- And administering an insulin bolus.

- Finally, we'll touch upon some useful tips and precautions.

***3.1 Programming basal insulin***

The mylife Ypsopump permits only two basal insulin rates - basal A and basal B. If a single basal rate suffices for you, simply set basal B to match basal A.

To program the basal rate, navigate to "basal rates" in the main menu and select Basal A. Adjust the insulin units per hour for the first hour of the day by scrolling, then confirm with the check mark. Repeat this process for all 24 hours. Swipe right to revisit the previous hour. Upon completion, the pump displays the total basal insulin dose over a day along with a graph of the scheduled basal rates. Confirm to save the settings. Under basal rates submenu, you'll now see a total daily basal insulin dose listed against Basal A.

As stated, Basal B must be set up similarly to Basal A. Select basal B in the main menu and program the basal insulin rate accordingly.

<span class="mark">If you</span> want to stop basal insulin delivery**<span class="mark">,</span>** this can be done via the main menu, opting for stop mode. Resuming insulin delivery involves the same steps.

A notification will pop up if the pump is in stop mode for over an hour. To receive this alert instantly, long-press the function key, and confirm.

***3.2 Temporary basal***

To start a temporary basal, visit the basal rates submenu and look for the icon symbolizing temporary basal. Here, you can set the rate (from 0 to 200%) and duration (15 minutes to 24 hours). Once confirmed, a percentage symbol appears next to the active basal profile (A or B) on the home screen. The basal rate alternates between the set percentage and the remaining duration of the temporary basal.

When the temporary basal expires, a message prompts for your acknowledgment. Regardless of your response, the pump defaults to the scheduled basal rate.

To cancel the temporary basal, access the basal rates submenu and select the temporary basal icon, which switches the pump back to the pre-set basal rate.

***3.3 Delivering a bolus***

To administer a bolus, access the main menu and click on the bolus icon represented by a knife and fork. A submenu presents three options: standard bolus, extended bolus, and combination bolus. The mylife YpsoPump lacks an inbuilt bolus calculator, so this setting isn't available here.

- For a standard bolus, you can select an amount between 0.1 and 30 units. An icon on the status screen denotes an active bolus, with a countdown showing the remaining units to be delivered. The bolus can be canceled by pressing the cross.

- If you opt for an extended bolus, after setting the bolus units, specify the delivery duration (ranging from 15 minutes to 12 hours). The status screen will then display not only the remaining insulin units to be delivered but also the time left.

- For a combination bolus, input the total bolus units first, then specify the immediate units, and lastly, the duration for the remaining bolus delivery. The top-left icon indicates the active bolus settings.

***3.4 Do's & don’ts***

- The pump lacks a built-in bolus calculator. Hence, if you wish to administer an insulin bolus based on carbohydrate intake, it should be done via the mylife Loop app or the mylife Dose app.

- The maximum bolus on the mylife YpsoPump is fixed at 30 U.

- The mylife Ypsopump also features a blind bolus function to deliver a bolus without viewing the pump. How to activate this function and the steps involved can be found in the instruction manual.

- If you prefer a larger bolus increment, you can modify this. From the main menu, select the settings icon represented by a wrench, and choose Set bolus increment. You can adjust it to 0.1, 0.5, 1, or 2 units.

- At times, you may not be able to deliver a combination bolus. This occurs when the remaining insulin in the reservoir is less than twice the bolus increment.

And there you have it, a guide on how to program and modify insulin settings on the mylife YpsoPump. In our next video, we'll discuss how to replace the insulin reservoir and infusion set.
