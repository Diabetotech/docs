## **<u>3. Insulin setup</u>** 

Welcome to this comprehensive video guide on setting insulin delivery using the MiniMed 780G pump.

In this segment, we will cover programming basal insulin, setting temporary basals, delivering a bolus with or without the bolus wizard, and important do's and don'ts to keep in mind.

### ***3.1 Programming basal insulin*** 

Let's begin by exploring basal insulin programming. To adjust basal insulin levels, navigate to the pump menu and select Insulin or Settings. From there, choose "Delivery Settings" and "Basal Pattern Setup" to specify the pattern you wish to modify. Under the options, you can edit the hourly blocks and insulin rates. Always begin at 0 hours and end at 24 hours. For example, you can set a basal rate of 0.9 units per hour from 0 to 8AM, 0.65 units per hour from 8AM to 6PM and 0.9 units per hour until midnight. Once you have made the desired adjustments, save the changes.

To stop basal insulin, go to the insulin bottle in the menu or use the up arrow for quicker access. Choose "Suspend All Delivery" to stop insulin delivery and click yes to confirm. The home screen will show "Delivery Suspended," and you'll receive reminders every 15 minutes. To resume, click on the up button, select "Resume Basal" and confirm to restart insulin delivery.

### ***3.2 Temporary basal***

When you need to administer a temporary basal, select the insulin bottle in the menu and select "Basal" followed by "Temp Basal." Adjust the duration and percentage as needed. Alternatively you can also choose a specific basal rate. While the temporary basal is active, you will see a green banner displayed on the home screen, indicating its remaining duration. To cancel a temporary basal, return to the insulin bottle in the menu, where you will find the option to cancel the temp basal. This process is intuitive and self-explanatory.

### ***3.3 Delivering a bolus*** 

Next, we'll delve into setting up the bolus calculator, known as the bolus wizard at Medtronic. To configure it, go to the menu, and select settings. Select "delivery Settings" and then select "Bolus Wizard Setup". Select "Bolus Wizard" to turn on the feature. Here it is shown how you adjust the necessary parameters for the bolus wizard, namely the carbohydrate ratio, the insulin sensitivity, the target glucose and the active insulin time.

Now, let's discuss giving a bolus. We will cover two methods: manual bolus and bolus using the bolus wizard.

- For a manual bolus, access the insulin bottle in the menu and select the "bolus" option or use the down arrow as a shortcut. The down arrow serves as a shortcut to the bolus screen, where you can enter the desired insulin amount. For instance, you can give a bolus of 0.5 units using this method.

- To give a meal bolus using the bolus wizard, select the insulin bottle, choose "give bolus," and activate the bolus wizard. The wizard will consider factors such as blood glucose levels, carbohydrate intake, and active insulin to calculate the recommended bolus amount. Simply enter the relevant information, and the wizard will present the suggested bolus. Confirm and deliver the bolus as instructed.

In the manual mode of the MiniMed 780G pump, you can choose between 3 types of bolus deliveries, including normal bolus, Square Wave bolus, and Dual Wave bolus. Square and Dual Wave boluses can be set up and delivered only after the Dual/Square Wave feature has been turned on in the Delivery Settings.

### ***3.4 Do's & don’ts***

Before we conclude, let's go over some important do's and don'ts:

- Ensure maximum bolus limits are set slightly above expected bolus doses. You can set this betwee<span class="mark">n 0 and 25 units. </span>

- <span class="mark">Consider adjusting the Bolus increment in the Delivery Settings to 0.025 units for optimal SmartGuard performance.</span>

- <span class="mark">The bolus speed determines how quickly the pump delivers bolus insulin. You can choose a standard rate (1.5 units per minute) or a quick rate (15 units per minute). If you’re experiencing a lot of infusion set blockages with the quick rate, it might be helpful to switch to standard rate.</span>

- <span class="mark">The MiniMed pump offers an Easy bolus feature, although it is not commonly utilized. This feature allows you to deliver a regular</span> bolus using only the up button. For more information, please refer to the pump's manual.
