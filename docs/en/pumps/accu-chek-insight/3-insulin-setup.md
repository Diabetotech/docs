## **<u>3. Insulin setup</u>**

Welcome to this instructional video on how to set insulin delivery on the Accu-Chek Insight insulin pump.

Before we begin, please note that if you are using this pump in conjunction with Diabeloop, insulin delivery will be controlled through the handset. You will learn more about that in the Diabeloop chapter. However, it's useful to know how to control the pump directly from the pump itself in case you forget your handset.

In this video, we will cover the following topics:

- Programming basal insulin

- Setting temporary basals

- Administering an insulin bolus

- And important do's and don'ts

### ***3.1 Programming basal insulin***

If you are using the Accu-Chek Insight pump with Diabeloop, basal insulin is typically set on the Diabeloop handset. However, you can still check the set basal profile from the pump itself. Simply navigate to the menu, select "Basal," and activate the desired basal profile. The current profile, usually set to DBLG1, corresponds to the basal security profile on your handset. Pressing it will display the profile details.

To temporarily stop insulin delivery, go to the menu, select "Stop Pump," and confirm. The status screen will then indicate "Pump stopped," and the status icon will change to a red square.

In stop mode, the pump will beep every minute. You can silence this stop warning by pressing either of the top buttons for 3 seconds.

To restart the pump, go to the menu, select "Start Pump," and press OK. The status icon will change back to a green triangle, indicating that the pump has restarted.

### ***3.2 Temporary basal***

To set a temporary basal, access the menu, choose "Basal," and select "Temporary Basal Dose." Choose the desired basal TBD (to be determined) option and use the **+ and - buttons** to set the percentage and duration of the temporary basal insulin. Confirm with OK. The status screen will display information about the temporary basal dose.

To cancel a temporary basal, navigate to the menu, select "Basal," choose "Temporary Basal Dose," and then "Cancel TBD." Confirm with OK, and an orange warning will appear to indicate the cancellation. Press OK again to finalize.

### ***3.3 Delivering a bolus***

To give a standard insulin bolus, access the menu, select "Bolus," and choose "Standard Bolus." Use the up and down arrow buttons to select the desired units and confirm with OK three times. The status screen will show the progress of the bolus.

In the Bolus submenu, you can also choose for an extended or a multiwave bolus.

Note that the Accu-Chek Insight pump does not have a bolus calculator. If you want to calculate a bolus based on carbohydrate intake, it must be done through Diabeloop's handset, the Accu-Chek Aviva Insight Diabetes Manager or the MySugr app.

To cancel a bolus, navigate to the menu, select "Bolus," and choose "Cancel Bolus." During a bolus, pressing either of the top buttons for 3 seconds will also cancel the bolus.

### ***3.4 Do's & don’ts***

- The pump lacks a built-in bolus calculator. Hence, if you wish to administer an insulin bolus based on carbohydrate intake, it should be done via the Diabeloop handset, the Accu-Chek Aviva Insight Diabetes Manager or the MySugr app.

- Ensure maximum bolus limits are set slightly above expected bolus doses. You can set this between 0 and 50 units. 

- The Accu-Chek Insight insulin pump also has a Quick Bolus feature. This lets you discretely deliver a bolus while your pump is hidden in your clothing. You can find more information about this in the manual.

Congratulations! You now know how to set and adjust insulin delivery on the Accu-Chek Insight pump. In the next video, we will demonstrate how to change the insulin reservoir and infusion set.
