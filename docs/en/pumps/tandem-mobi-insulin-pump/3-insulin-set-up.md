## **<u>3. Insulin set-up</u>**

Welcome to our detailed guide on insulin delivery set-up for the Tandem <span class="mark">Mobi</span> insulin pump.

We'll explore how to program basal insulin, set a temporary basal rate, and administer an insulin bolus. Additionally, we'll share some crucial do's and don'ts to enhance your pump experience.

### ***3.1 Programming basal insulin*** 

- To access your personal profile tap ‘settings’ from the navigation bar, next tap ‘Pump’, tap ‘Personal profiles’, tap ‘Add New Profile’ then tap the ‘Profile Name field’.

- Enter a profile name up to 16 characters and tap ‘next’. Tap ‘Add Segment’ to begin setting insulin delivery settings. The first Time Segment will start at 12 AM.

- Within each time segment, you must program a basal rate, correction factor, carb ratio and target BG. To enter values for each variable, tap the area labeled “Tap to set up”. A number keypad will appear, and once you have entered the first one, simply tap the field to enter the next value. Tap Done to close the keypad.

- Review all of the values you just entered and tap ‘Save’. A Time Segment has been saved banner is displayed at the top of the Tandem Mobi mobile app.

- If you need to add more time segments tap the plus icon. When adding more time segments, any settings that you entered in the previous time segment are copied and appear in the new segment. This allows you to simply adjust only the specific settings you want rather than having to enter them all over again.

- Tap ‘Add Segment’ then tap ‘ Start Time’. Using the on-screen keypad enter the time that you want the time segment to begin and tap’ Done’. Remember that this segment will be active from the start time until 12 AM or until the next segment you enter.

- When you have confirmed your settings and are done adding time segments, tap the back arrow icon to program your bolus settings. First tap the ‘Bolus Settings’ panel to get started. <span class="mark">By default the insulin duration is set at 5 hours and the carbohydrates are turned on. We recommend to leave these settings on the default values.</span>

- To delete a time segment, tap the start time of the time segment you wish to delete and tap ‘Delete Segment’. Tap ‘Yes’ to confirm.

- If you want to Rename an existing Profile, from navigation bar, tap ‘ setting’. Tap ‘ Pump’, then ‘Personal Profiles’. Tap the name of the Personal Profile to be renamed. Tap ‘Profile Options’ then tap ‘ Rename’.

- When you have confirmed your settings, tap the ‘Dashboard’ to return to the home screen. If you need to add additional personal profiles repeat the steps outline in this video.

- You may also duplicate an existing personal profile. Then select edit to change only certain settings to create a new profile.

To stop basal insulin, from the navigation bar, tap ‘Actions’ then select 'Stop insulin' and tap ‘Yes’. When insulin is stopped, the home screen will show the status “All deliveries stopped”. Restarting basal insulin can be done in the same way via ‘Actions’.

### ***3.2 Temporary basal***

To start a temporary basal, navigate to 'Actions' from the navigation bar and select 'Temp Rate'. Here, you can set the percentage and duration of your choice. Tap Dashboard on the Navigation bar to view the Dashboard screen and confirm the icon indicating a Temp Rate is active. A 'T' in a yellow box means Temp Rate is Active, while a ‘T’ in a red box means Temp Rate of 0 u/hr is active. You can always manually turn off the temporary basal, which reverts the 'T' back to the current system settings.

***3.3 Delivering a bolus***

Administering a bolus is straightforward. From the start screen, hit the bolus button, input the carbohydrate grams and, if necessary, a finger prick value. If paired with a glucose sensor, the last glucose value will auto-fill. The pump recommends an insulin dose, which you can override.

A handy tip is the option of extended bolus delivery. When confirming a bolus, you can opt to make it an extended bolus and adjust the percentage and time duration as needed.

### ***3.4 Do's & don’ts***

- Ensure maximum bolus limits are set slightly above expected bolus doses. You can find the Max Bolus via the pump settings. Tap ‘Delivery Limits’, then tap ‘Max Bolus’. The maximum bolus defaults to 10 units, but you can set it between 1 and 25 U.

- <span class="mark">It's recommended to set the Target BG value at 110 mg/dl or 6.5 mmol/l, as this is consistent with the target used when using Control-IQ technology</span>**.**

- It is recommended that all users set up the Quick Bolus function which enables you to deliver a bolus by simply pressing the pump button. It is a way to deliver a bolus by following beep/vibration commands without navigating through or viewing the pump screen.

And that's it! With this knowledge, you can effectively set-up and adjust insulin on the Tandem Mobi Insulin pump. In our next video, we'll walk you through changing the insulin reservoir and infusion set.
