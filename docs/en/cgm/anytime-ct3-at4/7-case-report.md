## **<u>7. Case report (2.5 min)</u>**

# Congratulations on completing this module on the Anytime CT3 and AT4 CGMs!

# You now understand how the sensor is inserted, how the transmitter functions, how the app is set up, and how glucose data can be reviewed and interpreted within the platform.

# In this final lesson, we shift from technical explanation to real-world user experience.

# We refer to an [<u>independent report by Lukas from OpenSource Clinic</u>](https://www.opensource.clinic/blog/details/Insights-from-Testing-10-CGMs---How-They-Stack-Up-in-Daily-Use), who wore the Anytime CT3 alongside several other CGM systems in daily life. His article describes hands-on, day-to-day experience. It is important to be clear: this is a personal user review. It is not a clinical trial, not a validation study, and not accuracy research.

# What this type of report offers is practical insight into how the device feels and behaves in real use.

# In his testing, Lukas describes the general wear experience — how the sensor fits into normal daily routines, how it behaves during regular activity, and how comfortable it is over the wear period. He notes that although the Yuwell sensor is the largest and thickest among the devices he wore, it remains comfortable because of its rounded design.

# He also reflects on how the glucose curves appear during everyday life. This includes observations about trend visualization and how intuitive the data feels when reviewing patterns over time.

# Another important aspect covered in the report is interaction with the software environment. This includes the setup process, navigation within the app, and the way glucose reports are structured and displayed. For educational purposes, this is relevant because usability directly affects how easily people can review and understand their glucose data.

# Finally, the report includes a comparative personal impression of the CT3 when worn alongside other CGM systems. Again, this reflects subjective experience — not measured performance — but it provides context on how the device integrates into daily use.

# When reviewing independent user reports like this, it is essential to separate experiential feedback from clinical evidence. User experience informs comfort, practicality, and workflow. Clinical data informs medical decisions. Both perspectives are useful, but they are not interchangeable.

# You have now completed the Anytime CT3 and AT4 module.

# Understanding a CGM system means understanding not only how it works technically, but also how it is experienced in everyday life.

# Well done.
