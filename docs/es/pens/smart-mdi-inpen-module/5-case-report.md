## **<u>5. Informe de caso</u>**

Para concluir este módulo, presentamos un ejemplo real de cómo una persona con diabetes tipo 1 experimenta el sistema Smart MDI, que combina la pluma de insulina inteligente InPen y el medidor continuo de glucosa Simplera CGM. La reseña, titulada «Probé la pluma de insulina inteligente: reseña de Simplera Smart MDI», está disponible en YouTube y presenta a una persona con diabetes tipo 1 que pasó de una bomba de insulina al sistema Smart MDI. Ver el vídeo El revisor destaca varios aspectos positivos del sistema, como la reducción de la carga mental, los recordatorios útiles, la mejora de la conciencia glucémica y una mayor confianza. Aunque el sistema fue bien recibido en general, el revisor señaló algunas áreas que podrían mejorarse.

- Gestionar dos aplicaciones distintas, una para la glucosa (Simplera) y otra para la insulina (InPen), resultaba a veces engorroso.

- La funcionalidad completa solo conjuntamente Simplera CGM e InPen.
