## **<u>1. MiniMed 780G:</u>**

- Características: el MiniMed 780G tiene un algoritmo muy bueno con bolos de autocorrección cada 5 minutos y con un alto tiempo en rango en estudios pivotales y del mundo real. Es el único sistema que ofrece un set de infusión de 7 días.

- Consideraciones: Las desventajas son que el sensor Guardian podría no durar los 7 días completos, y toma algún tiempo para reemplazar, aunque Medtronic también ofrece un sensor Simplera Sync e Instinct que son mucho más fácil de usar. La bomba MiniMed también es bastante grande.
