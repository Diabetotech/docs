## **<u>3. Recomendaciones de conducta</u>**

- Utilice el informe diario para comprobar el bolo previo a las comidas, el momento de introducción de carbohidratos, la anulación de las recomendaciones de bolo y el uso o los ajustes preestablecidos para el entrenamiento y antes de las comidas.

- Confirme que el tiempo de uso del sensor y el tiempo en el modo automatizado sean constantes, ya que las diferencias pueden indicar problemas con el sensor o la conexión.

- Revise el uso de las alarmas y los recordatorios para evitar la fatiga por alarmas.
