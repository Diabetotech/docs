## **<u>5. Compartir datos</u>**

<span class="mark">Bienvenido a esta lección sobre cómo compartir datos con el sensor FreeStyle Libre 2.</span>

<span class="mark">En este capítulo, exploramos el proceso de compartir datos con su proveedor sanitario y sus seres queridos, y de crear informes. 5.1 Compartir datos con su proveedor de asistencia sanitaria</span>

<span class="mark">Su proveedor sanitario accede a sus datos a través de LibreView, el software de informes y lectura de Abbott. Existen versiones para pacientes y profesionales, y puede crear su cuenta en libreview.com en cualquier momento.</span>

<span class="mark">En LibreView para profesionales, puede crear su consulta y obtener un número de identificación. Este número es necesario para enlazar con las cuentas personales de LibreView de sus pacientes. Tiene la opción de cambiar este número de identificación más adelante para recordarlo más fácilmente y compartirlo con los pacientes.</span>

<span class="mark">Los datos pueden cargarse en LibreView de dos maneras, dependiendo del dispositivo de escaneado:</span>

<span class="mark">●​ Si utiliza el lector, es necesaria una carga manual. El lector puede almacenar hasta 3 meses de datos. Para cargar los datos, conecte el lector al ordenador mediante un cable USB y siga las instrucciones de LibreView. Puede hacerlo usted como usuario o su proveedor de asistencia sanitaria. Como profesional sanitario, puede cargar los datos con el nombre del paciente para facilitar su recuperación.</span>

<span class="mark">●​ Si utiliza su teléfono móvil, todos sus datos se enviarán automáticamente a LibreView una vez que se registre a través de la app LibreLink. En la app, también puede introducir el número de identificación de su consulta para unirse a ella. Una vez vinculado, es posible realizar un seguimiento remoto y en tiempo real de los pacientes.</span>

<span class="mark">5.2 Crear informes Una vez sincronizados todos los datos con LibreView, puede generar varios informes seleccionando "informes de glucosa". Entre los nueve informes disponibles, los más utilizados son:</span>

<span class="mark">●​ Informe AGP, con el perfil TIR, TBR y AGP de las dos últimas semanas.</span>

<span class="mark">●​ Registro diario, que ofrece información sobre el momento y el efecto de cada uno de los bolos de insulina en el día a día. ●​ Instantánea, que muestra la frecuencia de exploraciones diarias de los sensores. ●​ Y Detalles del dispositivo, donde se encuentran los ajustes de la alarma.</span>

<span class="mark">Encontrará orientaciones para la interpretación de los informes en nuestro módulo general sobre sensores de glucosa.</span>

<span class="mark">5.3 Compartir datos con la familia</span>

<span class="mark">Como familiar o cuidador, puede descargar la aplicación LibreLinkUp y conectarla a la cuenta personal de Libreview del usuario.</span>

<span class="mark">Para establecer la conexión: ●​ El usuario debe aprobar el uso compartido de datos accediendo al menú de aplicaciones conectadas de la aplicación LibreLink y seleccionando LibreLinkUp. Añada el nombre y la dirección de correo electrónico del seguidor y, a continuación, pulse añadir. ●​ A continuación, el seguidor puede descargar la aplicación LibreLinkUp, crear una cuenta y seguir las instrucciones proporcionadas.</span>

<span class="mark">●​ El propietario de la aplicación LibreLinkUp puede configurar sus propias notificaciones.</span>

<span class="mark">La adopción de estas funciones de uso compartido mejora la colaboración y la toma de decisiones informadas para un viaje más empoderado hacia una mejor salud.</span>
