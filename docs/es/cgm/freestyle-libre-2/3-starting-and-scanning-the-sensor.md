## **<u>3. Puesta en marcha y exploración del sensor</u>**

<span class="mark">Bienvenido a este vídeo, donde le mostraremos cómo iniciar y escanear los sensores FreeStyle Libre 2 y FreeStyle Libre 2 Plus. Puede hacerlo utilizando una aplicación en su teléfono o con un lector FreeStyle Libre 2. Para simplificar, solo hablaremos de la aplicación LibreLink. Sin embargo, en algunas regiones, la aplicación específica para el sensor FreeStyle Libre 2 o Libre 2 Plus puede aparecer como la aplicación «FreeStyle Libre 2» o simplemente como la aplicación «Libre».</span>

<span class="mark">Si utiliza un sensor FreeStyle Libre 2 normal y tiene tanto el lector como la aplicación LibreLink en su teléfono, puede elegir qué dispositivo utilizar para escanear. Incluso puede utilizar ambos dispositivos simultáneamente iniciando y escaneando primero el sensor con el lector y luego con su teléfono. Sin embargo, las alarmas sólo se enviarán al primer dispositivo con el que escanee.</span>

<span class="mark">Si utiliza un sensor FreeStyle Libre 2 Plus o tiene acceso a una versión actualizada de la aplicación LibreLink, puede controlar su glucosa en tiempo real en su teléfono mediante la aplicación FreeStyle LibreLink sin necesidad de escanear cada vez. Sólo es necesario escanear una vez para iniciar el sensor. Si sólo tiene acceso a una</span>

<span class="mark">FreeStyle Libre 2, puedes seguir utilizándolo, pero funcionará como un monitor de glucosa flash, lo que significa que tendrás que escanear el sensor cada vez para ver tus niveles de glucosa.</span>

<span class="mark">3.1 Lector FreeStyle Libre 2</span>

<span class="mark">Si opta por utilizar el lector FreeStyle Libre 2, veamos cómo funciona.</span>

<span class="mark">Y en la parte inferior, hay un puerto para tiras reactivas, que te permite medir la glucosa o las cetonas en sangre con un simple pinchazo en el dedo.</span>

<span class="mark">El lector FreeStyle Libre 2 es compatible tanto con las tiras reactivas de glucosa en sangre FreeStyle Precision como con las tiras de cetonas de Abbott. Al insertar la tira reactiva y aplicarle sangre, puede utilizar el medidor de glucosa en sangre o el medidor de cetonas incorporados.</span>

<span class="mark">No olvides cargar el lector a diario, preferiblemente con el cable suministrado. A muchos usuarios les resulta cómodo cargarlo durante la noche junto a la cama, lo que garantiza la ininterrupción de las alertas mientras se carga. Tenga la seguridad de que, si carga el lector, podrá seguir utilizándolo. Ahora, profundicemos en el proceso paso a paso para escanear el sensor utilizando el lector: ●​ Pulse el botón azul de inicio para encender el escáner.</span>

<span class="mark">●​ Una vez encendida, la pantalla de inicio muestra información esencial como la hora actual, el nivel de batería, la hora de finalización del sensor y un icono de rueda dentada para acceder a los ajustes en la parte superior. En el centro de la pantalla, encontrarás dos opciones: "Comprobar glucosa" y "Revisar historial". Si no tienes un sensor activo, la primera opción será "Iniciar nuevo sensor".</span>

<span class="mark">●​ Pulse "Iniciar nuevo sensor" y mantenga el lector a menos de 4 centímetros (o 1,5 pulgadas) del sensor para iniciar la exploración. Si los sonidos están activados, el lector emite un pitido al activarse el sensor.</span>

<span class="mark">●​ Aproximadamente 1 hora después de la puesta en marcha, podrás ver los datos de glucosa de tu sensor cada vez que lo escanees. Para escanear el sensor, pulse el botón azul para iniciar el lector y simplemente mantenga el lector a menos de 4 centímetros (o 1,5 pulgadas) del sensor de nuevo. Puede escanear a través de tu ropa. Si los sonidos están activados, el lector emite un pitido tranquilizador una vez que ha escaneado correctamente el sensor. Mientras escanea, el lector mostrará el nivel de glucosa actual del sensor y una flecha de tendencia de glucosa.</span>

<span class="mark">●​ Si su nivel de glucosa es inferior a 40 mg/dL o 2,2 mmol/L, el lector mostrará "LO" Por el contrario, si supera los 500 mg/dL o 27,8 mmol/L, mostrará "HI". En estos casos, es importante verificar el nivel de glucosa con un pinchazo en el dedo y tomar las medidas adecuadas para tratar los niveles de glucosa bajos o altos. ●​ La esquina superior izquierda de la pantalla proporciona información adicional sobre su nivel de glucosa.</span>

<span class="mark">●​ Pulsando el icono del lápiz en la esquina superior derecha, puedes añadir detalles importantes como la dosis de insulina, la ingesta de alimentos o el ejercicio. Recuerde pulsar "Aceptar" para guardar la información. Esta acción se reflejará con un pequeño marcador, como una jeringuilla de insulina o una manzana, en el gráfico del sensor. ●​ El gráfico del sensor muestra sus niveles de glucosa de las últimas 8 horas. Si esperas más de 8 horas para escanear tu sensor, habrá un hueco en la información del escaneo.</span>

<span class="mark">Para explorar su historial, vaya a Revisar historial en la pantalla de inicio. Aquí encontrará el registro, el gráfico diario, la glucosa media, las tendencias diarias, el tiempo dentro del intervalo, los eventos de glucosa baja y el uso del sensor.</span>

<span class="mark">La pantalla de inicio también incluye un botón de configuración, que te ofrece más opciones de personalización. Dentro de los ajustes, puedes:</span>

<span class="mark">●​ ajustar alarmas</span>

<span class="mark">●​ ajustar las preferencias de sonido y vibración</span>

<span class="mark">●​ ajustar la hora y la fecha ●​ establecer recordatorios</span>

<span class="mark">●​ probar el medidor de sangre o de cetonas incorporado utilizando una solución de control</span>

<span class="mark">●​ cambiar el idioma ●​ ver el estado del sistema</span>

<span class="mark">●​ cambiar la configuración del informe</span>

<span class="mark">●​ consultar los puntos básicos del lector</span>

<span class="mark">Por último, también existen opciones profesionales. Para acceder a ellas, debes confirmar que eres un profesional sanitario e introducir el código CAA1C. Una vez dentro, podrás modificar el incremento de la dosis de insulina y configurar una calculadora de bolos.</span>

<span class="mark">No profundizaremos en cómo utilizar la calculadora de bolo incorporada, ya que la calculadora sólo se puede utilizar con los resultados de glucosa en sangre obtenidos con la punción digital del medidor incorporado. No puede utilizar la calculadora de insulina con las lecturas de glucosa del sensor. 3.2 Aplicación LibreLink</span>

<span class="mark">Ahora, vamos a explorar el proceso de iniciar y escanear el sensor utilizando la aplicación LibreLink en su teléfono móvil.</span>

<span class="mark">El primer paso es descargar la aplicación LibreLink, disponible tanto para dispositivos Android como iPhone.</span>

<span class="mark">El uso de la aplicación LibreLink no tendrá ningún efecto perceptible en la duración de la batería de su teléfono durante un uso normal. Incluso si la batería de su teléfono se agota, no se preocupe: sus datos permanecerán seguros. Puede escanear el sensor o recibir datos de glucosa con total confianza mientras el teléfono se carga.</span>

<span class="mark">Al iniciar la aplicación, observará un icono de menú de hamburguesa en la parte superior izquierda. En algunos teléfonos Android, parte de la configuración está disponible en un menú adicional de desbordamiento en la esquina superior derecha. Aquí puede ver qué funciones están disponibles a través del menú principal. Esta vista general de la pantalla de inicio y el menú de la aplicación FreeStyle LibreLink puede descargarse debajo de este vídeo</span>

<span class="mark">Iniciar y escanear el sensor es un poco diferente entre la versión Android e iOS de la aplicación LibreLink:</span>

<span class="mark">●​ En un iPhone, debe pulsar "Iniciar nuevo sensor" o "Escanear" antes de escanear el sensor.</span>

<span class="mark">●​ En un dispositivo Android, no tienes que pulsar ningún botón para hacerlo. Basta con abrir la aplicación LibreLink y escanear el sensor para iniciarlo.</span>

<span class="mark">●​ Es importante tener en cuenta que la ubicación de la antena NFC (Near Field Communication) difiere entre los dispositivos iPhone y Android.</span>

<span class="mark">●​ En un iPhone, normalmente se encuentra en la parte frontal del dispositivo, a menudo en la parte superior central.</span>

<span class="mark">●​ En los dispositivos Android, suele estar en la parte trasera, en el centro o en la parte superior. ●​ Tómese un momento para mover suavemente el teléfono sobre el sensor, observando dónde está situada la funcionalidad de escaneo. Familiarícese con la posición de la antena NFC para garantizar el éxito de los escaneos.</span>

<span class="mark">●​ Si tiene problemas con el escaneado, es posible que la antena NFC esté dañada o que la función NFC esté desactivada. Asegúrate de comprobar estos ajustes antes de continuar con la solución de problemas.</span>

<span class="mark">●​ Algunas fundas de teléfono bloquean el NFC, impidiéndo el escanear. Si la carcasa de su teléfono es de plástico, silicona u otro material no conductor, es probable que pueda escanear el sensor sin quitar la carcasa. Si la carcasa es de aluminio, metal u otro material con una superficie conductora, es posible que tenga que quitar la carcasa antes de escanear el sensor. El grosor de la carcasa también puede influir. Si la funda es muy gruesa, puede impedir que el teléfono escanee el sensor.</span>

<span class="mark">Una hora después de activar el sensor FreeStyle Libre 2 escaneándolo, podrás ver tus niveles de glucosa en la app LibreLink.</span>

<span class="mark">-​ En el caso del sensor FreeStyle Libre 2 normal, tendrás que escanear el sensor cada vez para ver tu nivel de glucosa. -​ Si utilizas el sensor FreeStyle Libre 2 Plus o tienes acceso a la app LibreLink actualizada, podrás controlar tu glucosa en tiempo real sin necesidad de escanear.</span>

<span class="mark">En la aplicación LibreLink, puede añadir fácilmente información adicional desde la pantalla de inicio, como su dosis de insulina, el número de carbohidratos que ha ingerido o el ejercicio que ha realizado.</span>

<span class="mark">La aplicación LibreLink también le permite añadir y ver notas en el diario en cualquier momento, lo que proporciona una mayor flexibilidad a la hora de registrar detalles importantes. Esto resulta más cómodo que en el lector, donde la toma de notas se limita a los 15 minutos posteriores a una lectura de glucosa.</span>

<span class="mark">Además, la aplicación ofrece funciones de informe más amplias, incluido el seguimiento en tiempo real de la HbA1c estimada. Para utilizar esta función, necesitarás al menos 5 días de datos del sensor. 3.3 Qué hacer y qué no hacer</span>

<span class="mark">Ahora vamos a hablar de lo que hay que hacer y lo que no hay que hacer para utilizar el sistema:</span>

<span class="mark">●​ El sensor es resistente al agua hasta 1 metro (3 pies) durante 30 minutos. Ten en cuenta que nadar más allá de esta profundidad o duración puede afectar a su funcionalidad.</span>

<span class="mark">●​ El lector no es resistente al agua y tiene una garantía de 2 años.</span>

<span class="mark">●​ Recuerda que la hora y la fecha del lector no se ajustan automáticamente. Tendrás     que   actualizarlas   manualmente      a   través   de   los   ajustes, especialmente cuando cambies de zona horaria. Cuando se ajusten, aparecerá un símbolo de reloj en los gráficos. Por otro lado, si utiliza la aplicación LibreLink en su teléfono móvil, la hora se sincronizará automáticamente con la de su teléfono.</span>

<span class="mark">●​ El lector puede mostrar diferentes mensajes de error, algunos con textos explicativos y otros con una E seguida de un número. Consulte el manual de usuario para obtener explicaciones detalladas de estos mensajes.</span>

<span class="mark">●​ Si utiliza Novopen 6 para inyectarse insulina, puede cargar directamente los datos de insulina en la aplicación LibreLink escaneando Novopen. Esta función requiere el escaneado para cargar los datos.</span>

<span class="mark">●​ Recuerde escanear el sensor cada 8 horas para garantizar una recopilación continua de datos. Si no lo hace, puede haber lagunas en los datos del sensor. Sin embargo, si utiliza FreeStyle Libre 2 Plus o tiene acceso a la aplicación LibreLink actualizada, no es necesario escanear el sensor después de iniciarlo.</span>

<span class="mark">●​ La aplicación LibreLink tiene funciones de texto a voz, que puede activar en los ajustes a través del menú de hamburguesa. En ese caso, podrá oír las lecturas de glucosa después de cada exploración, junto con la unidad de medida y la dirección de la flecha de tendencia.</span>

<span class="mark">Recuerde que, de vez en cuando, la tecnología puede presentarnos obstáculos inesperados. Si encuentra dificultades para escanear el sensor o conectar sus dispositivos, no se desanime. Sé persistente, explora las opciones de solución de problemas y ponte en contacto con el servicio de atención al cliente para obtener ayuda.</span>
