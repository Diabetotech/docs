## **<u>2. Colocación del sensor</u>**

Bienvenido a esta lección sobre la colocación del sensor Dexcom G7. En este vídeo, nos centraremos en colocar y retirar correctamente el sensor Dexcom G7 para garantizar lecturas precisas y un uso cómodo. Vamos a recorrer el proceso paso a paso.

### 2.1 Suministros Para colocar

correctamente el sensor, reúna los siguientes suministros: un aplicador de sensor con un sensor Dexcom G7 precargado y toallitas con alcohol para una higiene adecuada.

### 2.2 Colocación del sensor

Para insertar el sensor, primero elija el lugar de colocación. En EE.UU., el sensor Dexcom G7 está aprobado para su colocación en la parte posterior de la parte superior del brazo. Los niños de 2 a 6 años pueden colocar el sensor en las nalgas. En Europa, también está aprobado su uso en el abdomen tanto para niños como para adultos.

Antes de empezar, lávese y séquese las manos. Limpia tu zona con una toallita con alcohol y deja que se seque durante al menos 10 segundos. Coge tu aplicador y desenrosca el tapón. No toque el interior del aplicador. Presione y mantenga el aplicador firmemente contra su piel hasta que el protector de seguridad transparente se introduzca. Ahora presione el botón y retire el aplicador. Para ayudar a mantener el parche en su piel, frote firmemente alrededor del parche tres veces, luego presione suavemente el sensor durante 10 segundos. Vuelva a enroscar el tapón en el aplicador.

El sobreparche le ayudará a mantener el sensor colocado y se incluye con las instrucciones en la caja. Retire con cuidado los revestimientos transparentes de uno en uno. No toque la zona adhesiva blanca. Utilice la lengüeta de color para colocar el sobreparche alrededor del sensor. El uso de un espejo facilitará la applicación del parche. Frote alrededor del parche. Utilice la lengüeta para despegar el revestimiento de color y, a continuación, frote alrededor del sobreparche. Evite mojar el parche adhesivo, incluso en la ducha, durante las 12 horas siguientes a su applicación. 2.3 Retirada del sensor Cuando llegue el momento de retirar el sensor, despegue con cuidado el parche como si fuera una venda. Puedes utilizar aceite corporal o de bebé para ayudarte a retirarlo.

### 2.4 Qué hacer y

qué no hacer A continuación le ofrecemos algunos consejos clave que debe tener en cuenta:

- En la caja encontrarás varios números. El marcado con "21" es el número de serie, mientras que el número prefijado con "10" es el número de lote. Es posible que necesites estos números cuando pidas un sensor de repuesto. Así que es mejor que guardes la caja o le hagas una foto.

- En el aplicador encontrará un código de emparejamiento de 4 dígitos que necesitará para vincular el sensor al dispositivo receptor de datos que desee. Es una buena práctica conservar el aplicador o hacerle una foto para tenerlo como referencia. Una vez conectado, también puede encontrar el código de emparejamiento en la applicación Dexcom G7.

- Si alguna vez observa que el cable del sensor se desconecta de la aguja de inserción, esto indica un problema con el aplicador. Este problema puede deberse a que el aplicador se haya caído o dañado. En tales situaciones, se recomienda solicitar un sensor de repuesto, ya que de lo contrario la colocación del sensor no se realizará correctamente. No utilice el sensor dañado.

- Cuando coloque el sensor, no olvide presionar firmemente el aplicador contra la piel hasta que el protector de seguridad transparente se introduzca. De lo contrario, no será posible pulsar el botón blanco del aplicador.

- Si observa que el filamento del sensor sobresale del sensor después de la inserción, esto se denomina «cuello de cisne» y es un fallo conocido. El sensor no funcionará correctamente. Debe retirarse y puede solicitarse un reemplazo gratuito a Dexcom.

- Si su sistema Dexcom G7 detecta que el sensor ya no puede determinar su lectura de glucosa, puede recibir una alerta de Problema Breve del Sensor o una alerta de Fallo del Sensor. ○ La alerta de problema breve del sensor significa que el sensor es temporalmente incapaz de medir la glucosa. Si se produce este problema, suele ocurrir durante el primer día de sesión del sensor, pero puede ocurrir en cualquier momento. Suele solucionarse en 3 horas. Si continúa durante más de 3 horas, es mejor ponerse en contacto con el servicio de asistencia técnica. ○ El problema breve del sensor puede provocar una alerta de Sensor fallido. Esto significa que el sensor no puede medir la glucosa. Si recibe esta alerta, debe quitarse el sensor y comenzar una nueva sesión. ○ Para evitar futuros fallos del sensor, asegúrese de utilizar un lugar de inserción con suficiente grasa para evitar el tejido muscular. Asegúrese de limpiar la zona del sensor con jabón y una toallita con alcohol y deje que se seque completamente antes de colocárselo. Y asegúrese de utilizar el sobreparche Dexcom y de mantener el sensor bien adherido a la piel, para que el cable del sensor no se salga.

- Dexcom exige oficialmente el uso del sobreparche, pero no es necesario para todo el mundo, especialmente si no sudas mucho o piensas nadar con el sensor. Si no eres fan de los sobreparches Dexcom de color blanco, puedes comprar unos transparentes online o en la farmacia. Existen opciones de parches de calidad médica en tono o colores de piel que venden empresas como Skin Grip. Siguiendo estas pautas y siendo resistente ante los desafíos, podrá colocar y retirar su sensor Dexcom G7 con facilidad y confianza. El proceso de colocación del sensor es sencillo y no esperamos que surjan problemas. Sin embargo, si se encuentra con algún problema técnico, por favor no dude en ponerse en contacto con su soporte técnico local de Dexcom para obtener ayuda.
