## **<u>2. Colocación del sensor</u>**

Bienvenido a esta lección sobre cómo colocar el sensor Dexcom ONE Plus.

En este vídeo, nos centraremos en colocar y retirar correctamente el sensor Dexcom ONE Plus para garantizar lecturas precisas y un uso cómodo. Veamos el proceso paso a paso.

### 2.1 Suministros Para colocar

correctamente el sensor, reúna los siguientes materiales: un aplicador de sensores con un sensor Dexcom ONE Plus precargado y toallitas con alcohol para una higiene adecuada.

### 2.2 Inserción del sensor

Para insertar el sensor, primero elija el lugar donde lo colocará. El sensor Dexcom ONE Plus está aprobado para su colocación en el abdomen y la parte posterior del brazo. Además, los niños de 2 a 6 años también pueden colocarse el sensor en la parte superior de los glúteos.

Antes de comenzar, lávese y séquese las manos. Limpie la zona con una toallita con alcohol y déjela secar al aire durante al menos 10 segundos. Coge el aplicador y desenrosca el tapón. No toques el interior del aplicador. Mantén el aplicador firmemente presionado contra la piel hasta que se introduzca el protector transparente. A continuación, pulsa el botón y retira el aplicador. Para ayudar a que el parche se adhiera a la piel, frota firmemente alrededor del parche tres veces y, a continuación, presiona suavemente el sensor durante 10 segundos.

El parche le ayudará a mantener el sensor en su sitio y se incluye junto con las instrucciones en la caja. Retire con cuidado los protectores transparentes uno a uno. No toque la zona adhesiva blanca. Utilice la lengüeta de color para colocar el parche sobre el sensor. El uso de un espejo facilitará la aplicación del parche. Frote alrededor del parche. Utilice la lengüeta para despegar el revestimiento de color y, a continuación, frote alrededor del parche al menos tres veces.

Evite mojar el parche adhesivo, incluso en la ducha, durante las 12 horas posteriores a su aplicación. Transcurridas esas 12 horas, podrá ducharse y nadar con el sensor sin necesidad de protección adicional.

### 2.3 Retirada del sensor

Cuando llegue el momento de retirar el sensor, despegue con cuidado el parche como si fuera una tirita. Puede utilizar aceite corporal o aceite para bebés para facilitar la extracción.

### 2.4 Qué hacer y

qué no hacer Aquí hay algunos consejos clave que debes tener en cuenta:

- En la caja encontrará varios números. El marcado con «21» es el número de serie, mientras que el número precedido por «10» es el número de lote. Es posible que necesite estos números cuando solicite un sensor de repuesto. Por lo tanto, es mejor guardar la caja o hacerle una foto.

- En el aplicador encontrará un código de emparejamiento de 4 dígitos, que necesitará para vincular el sensor al dispositivo de recepción de datos que desee. Es recomendable guardar el aplicador o hacerle una foto para tenerlo como referencia. Una vez conectado, también podrá encontrar el código de emparejamiento en la aplicación Dexcom ONE Plus.

- Si observa que el cable del sensor se desconecta de la aguja de inserción, esto indica que hay un problema con el aplicador. Este problema puede deberse a que el aplicador se ha caído o se ha dañado. En tales situaciones, se recomienda solicitar un sensor de repuesto e informar del problema al servicio técnico de Dexcom, ya que, de lo contrario, la colocación del sensor no será satisfactoria. No utilice el sensor dañado.

- Al colocar el sensor, no olvide presionar firmemente el aplicador contra la piel hasta que se introduzca el protector transparente. De lo contrario, no será posible presionar el botón blanco del aplicador.

- Si observa que el filamento del sensor sobresale del sensor después de la inserción, esto se denomina «cuello de cisne» y es un fallo conocido. El sensor no funcionará correctamente. Debe retirarse y se puede solicitar un reemplazo gratuito a Dexcom.

- Si su sistema Dexcom ONE Plus detecta que el sensor ya no puede determinar su lectura de glucosa, es posible que reciba una alerta de «Problema breve del sensor» o una alerta de «Fallo del sensor».

- La alerta «Problema breve del sensor» significa que el sensor no puede medir la glucosa temporalmente. Si se produce este problema, suele ocurrir durante el primer día de una sesión del sensor, pero puede ocurrir en cualquier momento. Normalmente se soluciona por sí solo en 3 horas. Si continúa durante más de 3 horas, es mejor ponerse en contacto con el servicio técnico.

- En raras ocasiones, el problema del sensor breve puede provocar una alerta de fallo del sensor. Esto significa que el sensor no puede medir la glucosa. Si recibe esta alerta, debe retirar el sensor y comenzar una nueva sesión.

- Para ayudar a prevenir futuros fallos del sensor, asegúrese de utilizar una zona de uso con suficiente grasa para evitar el tejido muscular. Asegúrese de limpiar la zona del sensor con agua y jabón y una toallita con alcohol, y déjela secar completamente antes de insertarlo. Asegúrese también de utilizar el parche Dexcom y de mantener el

sensor bien adherido a la piel, para que el cable del sensor no se salga.

- Dexcom exige oficialmente el uso del parche adicional, pero puede que no sea necesario para todo el mundo, especialmente si no sudas mucho o si no tienes pensado nadar con el sensor. Si no te gustan los parches adicionales blancos de Dexcom, puedes comprar parches transparentes por Internet o en la farmacia. Existen parches de grado médico en tonos de piel o colores vendidos por empresas como Skin Grip.

Siguiendo estas instrucciones, podrá colocar y retirar el sensor Dexcom ONE Plus de forma fácil y segura. El proceso de colocación del sensor es sencillo y no prevemos que surja ningún problema. No obstante, si encuentra alguna dificultad técnica, no dude en ponerse en contacto con el servicio de asistencia técnica local de Dexcom para obtener ayuda.
