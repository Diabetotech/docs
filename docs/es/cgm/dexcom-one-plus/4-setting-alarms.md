## **<u>4. Configurar alarmas</u>**

Bienvenido a esta lección sobre cómo configurar alarmas con el sensor Dexcom ONE Plus.

A lo largo de esta sesión, le guiaremos a través de las opciones de alarma disponibles y le mostraremos cómo configurarlas en el receptor Dexcom ONE Plus y en la aplicación de su teléfono. ¡Empecemos!

### 4.1 Alarmas disponibles El

sistema Dexcom ONE Plus ofrece una serie de opciones de alarma para mantenerte informado sobre niveles específicos de glucosa y problemas relacionados con el sensor. Tienes la posibilidad de configurar alarmas para niveles bajos de glucosa, niveles altos de glucosa, pérdida de señal y problemas breves del sensor.

La alerta baja predeterminada se activa a 70 mg/dL o 3.9 mmol/L, mientras que la alerta alta predeterminada se activa a 250 mg/dL o 14 mmol/L, pero estos ajustes se pueden personalizar. Usted tiene control sobre el nivel de glucosa, el sonido y los intervalos de repetición de las alertas baja y alta. Para la alerta alta, hay una opción para retrasar la alerta inicial, que se activa solo si la lectura del sensor permanece alta durante un período prolongado.
Cuando reciba una alerta, es importante que tome medidas, como decidir el tratamiento o solucionar cualquier problema del sistema. Después, debe confirmar la alerta en su receptor, teléfono o reloj inteligente compatible pulsando «Aceptar». Hasta que no confirme la alerta, esta se repetirá cada 5 minutos.

### 4.2 Configuración de alarmas

en el receptor Dexcom ONE Plus Para configurar las alarmas en el receptor Dexcom ONE Plus, vaya a Configuración en el menú. Desde allí, seleccione la opción Alertas, donde podrá especificar las

diferentes alertas. Para cada alerta, puede especificar el nivel de alerta deseado y el intervalo de repetición cuando la condición persista.

Para personalizar los sonidos de alerta, acceda al menú Sonidos de alerta, donde encontrará varias opciones para la vibración, el volumen del sonido y las melodías de alarma.

Tenga en cuenta que el receptor debe permanecer encendido para recibir alarmas. Si el receptor está completamente apagado, no recibirá ninguna alerta.

### 4.3 Configuración de alarmas

en la aplicación Dexcom ONE Plus Con la aplicación Dexcom ONE Plus, puede acceder a la configuración de alertas en la pestaña Perfil. Para cada alerta, puede personalizar el nivel, el sonido y las preferencias de repetición. Para el sonido, puede optar por solo vibración o elegir entre diferentes melodías.

La aplicación también incluye algunas alertas técnicas poco frecuentes que no se pueden desactivar, pero para las que se puede elegir el sonido.

El uso eficaz de estos ajustes de alarma le permite mantenerse informado y ser proactivo en la gestión de sus niveles de glucosa con Dexcom ONE Plus. Al gestionar las alarmas, es esencial configurar solo las alertas que se adapten a sus necesidades y desactivar las que sean innecesarias. Las opciones de Dexcom para diferentes sonidos de alerta y alertas por vibración pueden ayudar a crear una experiencia más silenciosa. Además, estas alertas pueden contribuir a mejorar su tiempo en rango, ofreciendo un enfoque proactivo para responder a los cambios en los niveles de glucosa.
