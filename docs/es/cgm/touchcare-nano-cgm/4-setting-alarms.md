## **<u>4. Configuración de alarmas</u>**

Bienvenido. En esta lección hablaremos sobre la configuración de alarmas con el sensor TouchCare Nano.

En esta sección, le mostraremos las opciones de alarma disponibles y le explicaremos cómo configurarlas utilizando el PDM y la aplicación EasyPatch. Empecemos.

### 4.1 Alarmas disponibles El

MCG TouchCare Nano proporciona varias opciones de alarma para mantenerlo informado sobre sus niveles de glucosa y el estado del sensor. Entre ellas se incluyen alarmas para:

- Niveles bajos de glucosa

- Niveles altos de glucosa

- Sensor perdido, cuando se pierde la señal del sensor durante más de 10 minutos debido a problemas con el Bluetooth.

- Sin lecturas, cuando el sensor no puede detectar los valores de glucosa. Si el sensor no está bien insertado, sustitúyalo. De lo contrario, espere: es posible que se recupere. No retire el sensor a menos que reciba una alerta de Fallo del sensor.

Además, el sistema cuenta alertas predictivas para avisarle con antelación sobre posibles niveles altos o bajos de glucosa, y alertas de velocidad de cambio, que se activan cuando los niveles de glucosa suben o bajan rápidamente. Una alerta crítica es la alerta de glucosa baja por debajo de 54 mg/dl o 3.1 mmol/L, que es obligatoria y por su seguridad, no se puede desactivar.
También puede usar la función llamada "Repetición" para posponer las alertas que se repiten cuando los niveles de glucosa están constantemente altos o bajos. Esto le permite pausar temporalmente las notificaciones y evitar que suenen de forma continua.

Además, con la función de programación de alertas, puede personalizar los avisos según diferentes momentos del día, para que se adapten mejor a su rutina diaria.

Para asegurarse de recibir estas alarmas y alertas, mantenga el teléfono y el sensor a una distancia máxima de 10 metros (o 33 pies) el uno del otro sin obstáculos.

### 4.2 Configuración de alertas

en el PDM Para configurar las alertas de glucosa en su PDM, deslice para desbloquear su PDM. Deslice de derecha a izquierda para acceder al menú principal. Pulse sobre EasyLoop. Pulse sobre alertas de glucosa. Ahora active las alertas de glucosa. Pulse en límites de glucosa puede añadir hasta ocho segmentos horarios diferentes para el día en función de su necesidad de diferentes alertas de glucosa para diferentes momentos. Pulse añadir segmento de tiempo para añadir un nuevo segmento de tiempo. Puede ajustar los límites de glucosa para bajo y alto tocando los botones azules de bajo y alto. Una vez hecho esto, toque el icono de guardar en la esquina superior derecha del PDM para guardar sus límites de glucosa.

Desde el menú de alertas de glucosa:

- También puede configurar alertas predictivas para que se le avise de un valor previsto de glucosa alto o bajo con una antelación de entre 5 y 30 minutos.

- A través de las alertas de velocidad de cambio de glucosa, se puede establecer una alerta de descenso rápido o de aumento rápido.

- Y a través de la función Repetir, puede definir cada cuánto tiempo desea que se repita una alerta si los niveles de glucosa siguen siendo altos o bajos después de la primera notificación. Para las Alertas Altas, puede configurar que se repitan entre 5 minutos y 3 horas. Y para las Alertas Bajas, entre 5 minutos y 1 hora.

Las alertas pueden emitirse como sonidos, vibraciones o ambos. Para ajustar esta configuración, vaya al Menú principal, pulse Configuración, después General y seleccione Opciones de audio.

### 4.3 Configuración de alertas

en la aplicación EasyPatch Para configurar las alertas de glucosa en la aplicación EasyPatch, pulse sobre las tres líneas de la esquina superior izquierda para acceder al menú principal. Pulse sobre EasyLoop. Pulse sobre alertas de glucosa. Ahora active las alertas de glucosa. Pulse sobre límites de glucosa. Puede añadir hasta 8 segmentos horarios diferentes para el día en función de su necesidad de diferentes alertas de glucosa para diferentes momentos. Pulse Añadir segmento horario para añadir un nuevo segmento horario. Puede ajustar los límites de glucosa para su nivel bajo y alto pulsando en los botones azules de bajo y alto. Una vez hecho esto, pulse el icono de guardar en la esquina superior derecha de su dispositivo móvil para guardar sus límites de glucosa.

Desde el menú Alertas de glucosa, puede personalizar varias opciones de alerta:

- Establecer alertas predictivas de glucosa alta y baja, así como alertas de subida rápida y bajada rápida.

- Elija un tono específico para la alerta obligatoria de glucosa baja por debajo de 54 mg/dL o 3.1 mmol/L.

- Para otras alertas de glucosa alta y baja, puede configurar el tono, la opción "Repetir" y el tiempo de "Repetición": ○ La función "Repetir" controla cuánto tiempo debe pasar antes de que se emita una nueva alerta si no borra la anterior. ○ La función "Repetición" controla el intervalo entre alertas, incluso si no borra ni confirma la anterior. Esta función actúa como una especie de posponer.

Si gestiona estos ajustes cuidadosamente, podrá mantenerse informado y proactivo a la hora de mantener sus niveles de glucosa con el MCG TouchCare Nano. Este enfoque le permite responder rápidamente a los cambios y mejorar el control de su diabetes.
