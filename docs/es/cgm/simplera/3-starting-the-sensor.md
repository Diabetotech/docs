## **<u>3. Arrancar el sensor</u>**

<span class="mark">Bienvenido a esta lección sobre cómo iniciar el sensor Simplera.</span>

<span class="mark">En este módulo, le guiaremos a través del proceso de inicio del sensor Simplera utilizando la aplicación Simplera y la bomba de insulina MiniMed 780G.</span>

<span class="mark">3.1 Sensor Simplera con la aplicación Simplera La app Simplera está disponible para su descarga en dispositivos Android e iOS específicos. Puede comprobar la compatibilidad en línea en medtronic-diabetes.com.</span>

<span class="mark">Cuando abras la aplicación, una de las primeras cosas que tendrás que hacer es crear o iniciar sesión en tu cuenta CareLink. Una vez que hayas iniciado sesión, tendrás que aceptar un acuerdo de usuario. A continuación, la aplicación te guiará por algunos ajustes del teléfono. Te preguntará si quieres compartir tus datos y te enseñará a personalizar las alertas. A continuación, es el momento de insertar y emparejar el sensor. La aplicación te guiará por el proceso de aprendizaje. Cuando hayas insertado el dispositivo, pulsa "Lo he insertado" y la aplicación se conectará al sensor. Tras un tiempo de calentamiento de 2 horas, empezarás a obtener lecturas de glucosa cada 5 minutos. La pantalla de inicio muestra el valor de glucosa actual y el valor de tendencia, así como un gráfico del sensor.</span>

<span class="mark">Debajo hay 4 botones: ajustes, sensor, diario y eventos. -​ A través de los ajustes puedes personalizar las alertas y sincronizar los datos con CareLink y los cuidadores. -​ A través del sensor, puedes ver la vida útil de tu sensor. -​ Si tocas el sensor y luego opciones, puedes sustituir tu sensor o introducir una calibración opcional.</span>

<span class="mark">-​ A través de eventos puedes registrar diferentes eventos como glucosa en sangre, insulina, carbohidratos y ejercicio. Estos iconos se pueden encontrar en el gráfico del sensor y también se sincronizan con CareLink.</span>
