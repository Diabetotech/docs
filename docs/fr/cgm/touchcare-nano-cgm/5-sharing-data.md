## **<u>5. Le partage des données</u>**

Dans cette vidéo, nous allons vous expliquer comment partager vos données de
glycémie avec les professionnels de santé et les membres de votre famille qui vous
aident à gérer votre diabète.
Si vous utilisez l'application EasyPatch, vos données seront automatiquement
envoyées au nuage Medtrum toutes les 2 minutes. Si vous utilisez un GPD, vous
pouvez le connecter à votre smartphone à l'aide de l'application EasyTouch, ce qui
permettra le téléversement automatique des données de votre GPD vers l'application
EasyTouch toutes les 2 minutes. Cette application affiche les informations contenues
dans votre GPD et les envoie en parallèle vers le nuage.
5.1 Le partage des données avec votre professionnel de santé
Votre professionnel médical peut accéder à vos données glycémiques sur la plateforme
EasyView Pro, tandis que les patients et les soignants peuvent utiliser EasyView pour
surveiller et examiner les données du SCG. Les comptes EasyView (pour patient et
soignant) et EasyView Pro (pour professionnels de santé) peuvent être créés
gratuitement.
Pour commencer :
- Vérifiez que votre application EasyPatch ou EasyTouch est connectée à votre
capteur et transmet des données.

●​ Votre médecin peut vous ajouter en tant que patient à l'aide de votre adresse
e-mail enregistrée et vous inviter à vous connecter.
- Une fois que vous aurez accepté la demande de suivi, votre médecin pourra
suivre vos informations.
5.2 La création et la consultation de rapports
L'application EasyPatch met à la disposition de l'utilisateur un nombre de rapports.
Vous pouvez sélectionner les rapports quotidiens des capteurs et l'analyse des
tendances sur 7, 30, ou 90 jours.
Pour obtenir des rapports plus complets, vous pouvez vous connecter à la plateforme
EasyView. Plusieurs rapports sont alors disponibles :
- Le Tableau de bord, qui vous donne une vue d'ensemble et des moyennes sur
les données du patient.
- Le Résumé journalier, qui vous donne un aperçu détaillé d'une date
sélectionnée.
- La Vue d'ensemble jour par jour, qui vous donne les données des 7 ou 14
derniers jours sur une seule page.
- La Superposition des capteurs, qui vous donne les relevés SCG des sept
derniers jours, les uns au-dessus des autres.
- L'Analyse des tendances, qui vous donne des détails sur les moyennes de la
pompe et du SCG.
- Les Événements, qui vous donne l'historique de chaque événement enregistré
sur le téléphone du patient ou sur le combiné GPD.
- Le Journal de bord, qui fournit des détails sur les données des 7 ou 14 derniers
jours en ce qui concerne les débits de glucose, l'apport en glucides, les doses de
bolus ainsi que d'autres informations.
- Le rapport Alerte, qui vous donne un historique de toutes les alertes qui sont
apparues sur le smartphone ou l'appareil GPD, ainsi que sur la pompe.
- Enfin, l'examen des paramètres de l'appareil vous donne des détails sur les
réglages du capteur.
Ces rapports peuvent être imprimés, téléchargés ou partagés électroniquement, ce qui
vous aide, vous et votre équipe soignante, à prendre des décisions éclairées.

5.3 Le partage des données avec la famille et les soignants
La famille et les soignants peuvent utiliser l'application EasyFollow. Cette application
vous permet de partager vos données de glycémie en temps réel avec un nombre
illimité de soignants. Voici comment procéder :
- Pour se connecter, les aidants peuvent trouver le compte utilisateur en
saisissant leur adresse e-mail dans l'application EasyFollow ou sur le site web
EasyView.
- Pour accepter les demandes de suivi, l'utilisateur se rend ensuite dans Menu
principal → Paramètres → Sécurité du compte → Autorisation de visualisation à
distance dans l'application EasyPatch, ou dans Profil et connexions sur le site
web EasyView.
- Une fois connectés, les soignants peuvent consulter les mesures de glucose en
temps réel, les flèches de tendance et recevoir des alertes, avec la possibilité
d'ajuster les préférences de notification, tandis que l'utilisateur garde le contrôle
sur les données partagées.
Le partage des données avec les professionnels médicaux et les proches crée un
système de soutien qui améliore la gestion du diabète. En prenant cette mesure, vous
vous assurez que les personnes qui s'occupent de vous peuvent contribuer à votre
sécurité et à votre soutien.
