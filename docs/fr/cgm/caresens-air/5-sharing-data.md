## **<u>5. Leçon 5</u>**

5.1 Partage de données avec votre professionnel médical via l'application
Sens365
Les professionnels médicaux peuvent installer l'application Sens365 sur leurs
appareils afin de recevoir et de surveiller vos données glycémiques. Pour ce faire,
vous devez d'abord générer un code à six chiffres dans l'application CareSens Air.
Appuyez sur Paramètres en bas de l'écran d'accueil. Appuyez sur Prestataires de
soins. Appuyez sur Se connecter pour associer votre compte à celui de votre
professionnel de santé. Appuyez sur Générer un code de partage. Choisissez les
données que vous souhaitez partager avec votre prestataire de soins, puis appuyez
sur « Générer un code de partage ». Partagez ensuite le code généré qui s'affiche à
l'écran avec votre professionnel médical.
Entrons maintenant le code partagé dans l'application Sens365. Après avoir lancé
l'application Sens365 sur l'appareil du professionnel de santé, appuyez sur l'icône
représentant une personne en bas de l'écran d'accueil. Appuyez sur Ajouter.
Saisissez le code partagé fourni par un utilisateur du capteur et appuyez sur OK.
Vous pouvez également saisir le code partagé via les Paramètres en bas de l'écran
d'accueil. Appuyez sur Connexions. Appuyez sur l'icône Ajouter une personne dans
le coin supérieur droit de l'écran. Saisissez le code partagé fourni par un utilisateur.
Appuyez sur OK pour terminer le processus. Une fois le processus terminé, une
notification contextuelle apparaîtra sur votre application CareSens Air.
Vous pouvez également saisir le code fourni par votre professionnel médical dans
votre application CareSens Air pour vous connecter avec lui. Ouvrez l'application
CareSens Air et appuyez sur Professionnels Médicaux dans les Paramètres, puis
appuyez sur Se connecter. Appuyez sur Saisir le code de partage. Saisissez le code
fourni par le professionnel médical. Si le code est valide, une alerte de confirmation
de suivi par une professionnel médical s'affiche à l'écran. Assurez-vous d'ajouter le

bon professionnel de santé. Une fois le processus terminé, une notification
d'achèvement s'affichera.
Vous pouvez temporairement interrompre le partage des données de glycémie avec
votre professionnel médical ou modifier les éléments que vous partagez avec lui.
En bas de l'écran d'accueil, appuyez sur Paramètres, puis sur Professionnels
Médicaux. Sur l'écran, appuyez sur le professionnel dont vous souhaitez modifier
les options de partage.
Appuyez sur Partager les données pour activer ou désactiver le partage des
données. Cochez les cases pour arrêter le partage des données relatives aux
mesures de glycémie, aux événements ou aux alertes.
Si vous ne souhaitez plus partager vos données avec un professionnel médical,
vous pouvez désactiver le partage. Sur l'écran Professionnels Médicaux, appuyez
sur l'icône Supprimer à côté du nom du professionnel avec lequel vous souhaitez
cesser de partager vos données. Appuyez sur Déconnecter dans la fenêtre
contextuelle pour terminer le processus.
5.2 Partage de données avec votre professionnel médical via le site Web
Sens365
Toutes les données enregistrées par l'application CareSens Air sur votre appareil
sont automatiquement stockées sur le serveur cloud. Pour actualiser le
téléchargement des données de votre application, appuyez sur Paramètres en bas
de l'écran d'accueil, puis sur Télécharger.
Vous pouvez consulter et gérer vos données glycémiques téléchargées à l'aide de
la plateforme cloud Sens365 Web. Ouvrez votre navigateur Web et rendez-vous
sur https://sens365.i-sens.com. Cliquez sur « Commencer » sur l'écran d'accueil.
Lors de votre première visite sur le site, vous serez invité à choisir votre pays de
résidence et votre langue. Cliquez sur « Soumettre » pour confirmer vos choix.
Cliquez sur « Créer un compte » sur l'écran de connexion si vous n'avez pas encore
de compte Sens365. Entrez et vérifiez votre adresse e-mail. Acceptez les conditions
requises, créez votre mot de passe et remplissez toutes les informations requises.
Une fenêtre contextuelle « Inscription terminée avec succès » apparaîtra.

Si vous possédez déjà un compte Sens365, il vous suffit d'entrer votre adresse
e-mail et votre mot de passe enregistrés à l'écran pour vous connecter.
Lors de votre première connexion à Sens365 Web, l'écran de sélection du mode
utilisateur s'affichera. Si vous êtes un utilisateur CareSens Air, sélectionnez «
Utilisateur du capteur ». Si vous êtes un prestataire de soins, sélectionnez «
Prestataire de soins ».
Les utilisateurs du capteur doivent cliquer sur OK pour confirmer leur pays de
résidence et l'unité de mesure du glucose. Ils seront ensuite redirigés vers l'écran
d'accueil.
Pour vous connecter à un nouveau professionnel médical, cliquez sur Connexions
dans la barre de navigation à gauche. Cliquez sur le bouton Se connecter avec un
prestataire de soins à droite. Entrez le code de partage du soignant et cliquez sur
OK. Vérifiez le nom du groupe de partage. Après avoir lu l'accord, cochez la case et
cliquez sur OK. Une fois cette opération terminée, vous trouverez le nouveau
professionnel médical ajouté à la liste.
Pour vous déconnecter d'un professionnel de santé, cliquez sur le bouton
Déconnecter, puis cliquez sur Déconnecter dans la fenêtre contextuelle.
Passons maintenant au mode Soignant. Lorsque vous vous connectez, l'écran
affiche les noms des groupes et la liste des utilisateurs de capteurs dans chaque
groupe. Un résumé des informations de chaque utilisateur est également affiché.
Vous pouvez faire glisser une colonne pour la repositionner.
Lorsque vous cliquez sur le bouton Tableau de bord pour un patient spécifique, la
fenêtre contextuelle Ajouter au tableau de bord s'affiche. Attribuez un prestataire
de soins à l'utilisateur, puis cliquez sur OK. Le tableau de bord s'affiche alors dans
la partie droite de l'écran. Vous pouvez ouvrir le tableau de bord à tout moment en
cliquant sur le bouton Tableau de bord situé dans le coin supérieur droit.
Les groupes peuvent être créés ou supprimés. Pour créer un groupe, cliquez sur les
trois points à côté du nom du groupe. Dans la fenêtre contextuelle Paramètres du
groupe, personnalisez le nom du groupe et générez le code de partage du groupe,
puis cliquez sur Enregistrer.
Vous pouvez également ajouter un nouvel utilisateur ou en inviter un à rejoindre le
groupe. Si vous ajoutez un utilisateur de capteur, saisissez le code de partage que

celui-ci vous a communiqué. Si vous invitez un utilisateur, confirmez le nom du
groupe, puis envoyez votre code de partage à l'utilisateur. Vous pouvez copier le
code ou envoyer directement des invitations par e-mail. Le code de partage de
votre professionnel médical reste le même, sauf si vous le modifiez ; vous pouvez
générer un nouveau code de partage si nécessaire.
De plus, vous pouvez inviter un autre soignant à gérer les groupes d'utilisateurs
avec vous. Pour ce faire, cliquez sur le menu à trois points à côté du nom du groupe
et sélectionnez Inviter un professionnel médical. Saisissez l'adresse e-mail du
soignant, puis accordez le rôle Administrateur ou Coordinateur. Cochez la case de
confirmation et cliquez sur Envoyer.
5.3 Rapports
En mode Soignant, cliquez sur le nom de l'utilisateur du capteur pour accéder à
l'écran Reports. Par défaut, vous serez redirigé vers le rapport AGP. À gauche, vous
trouverez Aperçu. À droite, vous pouvez accéder à Comparaison, Modèles,
Tendances, Statistiques et Événements.
Dans la ligne ci-dessus, vous pouvez régler la période que vous souhaitez afficher.
La page Aperçu affiche Temps dans la plage, Glycémie moyenne, GMI, Variabilité
glycémique et Écart type. Glycémie moyenne quotidienne, insuline, Temps quotidien
dans les plages, Temps CGM actif et Informations sur l'appareil sont également
affichés. Une option de personnalisation est disponible, vous permettant d'ajuster
la mise en page en ajoutant, supprimant ou déplaçant des blocs.
AGP est un rapport glycémique standardisé qui comprend le temps dans la plage
cible, le temps d'activité du CGM, les mesures glycémiques, Profil ambulatoire de
glycémie et Profils quotidiens de glycémie. Vous pouvez vérifier le pourcentage de
temps passé dans la plage cible de glycémie à l'aide du graphique TIR, ou Temps
Dans la Cible, et estimer l'A1c à l'aide du GMI.
Sur la page Comparaison, deux graphiques sont affichés en fonction des périodes
sélectionnées. Cela permet de voir facilement les changements dans les tendances
glycémiques et les améliorations dans la gestion du glucose en comparant les deux
graphiques.
Sur la page Tendances, les plages de glycémie basse, cible et élevée sont indiquées
sur le graphique pour la période sélectionnée. Cela vous aide à identifier les

tendances en montrant quand les niveaux de glycémie sont bas ou élevés et quels
événements peuvent être associés aux variations de glycémie.
La page Tendances affiche les tendances quotidiennes et hebdomadaires de la
glycémie. Vous pouvez consulter les fluctuations quotidiennes ou voir toutes les
tendances quotidiennes réunies dans un graphique hebdomadaire.
La page Statistiques affiche les mesures de glycémie par jour de la semaine et par
heure de la journée. Vous pouvez consulter les statistiques quotidiennes et horaires
moyennes de glycémie pour la période sélectionnée.
La page Événements affiche tous les journaux d'événements enregistrés dans
l'application CareSens Air par ordre chronologique.
Plus haut sur l'écran, à côté du nom de l'utilisateur du capteur, cliquez sur
Paramètres de la plage cible pour régler la plage cible de glycémie pour les
périodes diurnes et nocturnes et adapter la gestion aux objectifs de l'utilisateur.
Dans les modes Utilisateur et Soignant, vous pouvez télécharger, imprimer et
envoyer des rapports, ainsi qu'exporter des données. Pour télécharger un rapport,
cliquez sur l'icône en forme de flèche vers le bas dans le coin supérieur droit ; le
rapport sera enregistré au format PDF. Pour imprimer, cliquez sur l'icône en forme
d'imprimante. Pour envoyer le rapport, cliquez sur l'icône en forme d'enveloppe ; il
sera envoyé au format PDF à l'adresse e-mail que vous aurez indiquée. Enfin, pour
exporter vos données au format CSV, cliquez sur l'icône Exporter les données.
Par ailleurs, sur le récepteur CareSens Air, l'option Statistiques du menu principal
vous permet de consulter les rapports des 7, 14, 30 ou 90 derniers jours. Chaque
rapport indique votre glycémie moyenne, l'écart type (ET) et le coefficient de
variation (CV) pour la période sélectionnée, ainsi que votre temps dans la plage
cible.
5.4 Conseils
Veuillez relire ces rappels avant de partager vos données CareSens Air :
- Lorsque vous vous connectez à un soignant via l'application CareSens, vous
disposez de 5 minutes pour saisir le code de partage généré. Si le code
expire, créez-en un nouveau.

●​ Lorsque vous modifiez vos options de partage pour un soignant, celui-ci est
informé des changements.
- Sens365 est un visualiseur secondaire pour le système CGM CareSens Air et
n'est pas destiné à fournir des décisions thérapeutiques ni à remplacer les
conseils d'un professionnel de santé. Sens365 ne garantit pas la réception
en temps réel des données glycémiques de l'utilisateur du CGM CareSens
Air.
- Vous ne pouvez pas partager directement les données depuis le récepteur
CareSens Air. Pour accéder aux données stockées sur l'appareil, rendez-vous
sur le site Web Sens365, connectez-vous et téléchargez les données du
récepteur vers le serveur cloud.
Dans cette vidéo, nous avons abordé différentes options pour partager et
télécharger vos données CareSens Air. Grâce à ces fonctionnalités, vous pouvez
collaborer plus étroitement avec votre équipe soignante, ce qui vous aide à prendre
des décisions éclairées en temps opportun pour améliorer votre santé.
