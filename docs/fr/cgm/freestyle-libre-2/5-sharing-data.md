## **<u>5. Partage des données</u>**

Bienvenue dans ce chapitre sur le partage des données avec le capteur FreeStyle Libre 2. Nous découvrons ensemble le partage des données avec votre médecin et vos proches, ainsi que la création de rapports.

### 5.1 Partager ses données avec son médecin

Votre prestataire de soins accède à vos données par l'intermédiaire de LibreView, le logiciel de rapport et de lecture d'Abbott. Des versions pour patients et pour professionnels sont disponibles. Vous pouvez créer votre compte sur libreview.com à tout moment. Dans LibreView pour les professionnels, vous pouvez créer votre cabinet et obtenir un numéro d'identification. Ce numéro est nécessaire pour établir un lien avec les comptes personnels LibreView de vos patients. Vous avez la possibilité de changer ce numéro d'identification plus tard pour faciliter la mémorisation et le partage avec les patients. Les données peuvent être téléchargées vers LibreView de deux façons, selon votre appareil de numérisation :

- Si vous utilisez le lecteur, un téléchargement manuel est nécessaire. Le lecteur peut stocker jusqu'à 3 mois de données. Pour télécharger les données, connectez le lecteur à votre PC à l'aide d'un câble USB et suivez les instructions de LibreView. Ce processus peut être effectué par vous en tant qu'utilisateur, ou par votre prestataire de soins de santé. En tant que fournisseur de soins de santé, vous pouvez télécharger les données sous le nom du patient pour les retrouver facilement.

- Si vous utilisez votre téléphone portable, toutes vos données seront automatiquement envoyées à LibreView une fois que vous vous serez inscrit via l'application LibreLink. Dans l'application, vous pouvez également entrer le numéro d'identification de votre cabinet pour le rejoindre. Une fois le lien établi, il est possible de suivre les patients en temps réel et à distance.

### 5.2 Créer des rapports

Une fois que toutes les données sont synchronisées avec LibreView, vous pouvez générer différents rapports en sélectionnant "Rapports de Glycémie". Parmi les neuf rapports disponibles, les plus souvent utilisés sont les suivants :

- Le rapport AGP qui relève la Durée dans les plages, la Durée en dessous des plages, et le profil AGP pour les deux dernières semaines.

- Le Journal Quotidien qui donne un aperçu du temps dans la cible et de l'impact des bolus d'insuline individuels sur le plan quotidien.

- L'Image instantanée qui affiche la fréquence des scans du capteur au quotidien.

- les Détails de l’Appareil où se trouvent les paramètres des alarmes. Un guide pour interpréter les rapports est disponible dans le module principal sur les capteurs de glucose.

### 5.3 Partager ses Données avec sa Famille

En tant que membre de la famille ou soignant, vous pouvez télécharger l'application LibreLinkUp et la connecter au compte personnel LibreView de l'utilisateur. Pour établir la connexion :

- Pour établir la connexion, l'utilisateur doit valider le partage des données en accédant au menu des applications connectées dans l'application LibreLink et en sélectionnant LibreLinkUp. Ajoutez le nom et l'adresse électronique du surveillant, puis appuyez sur ajouter.

- Le surveillant peut ainsi télécharger l'application LibreLinkUp, créer son compte et suivre les instructions fournies.

- L’utilisateur de l'application LibreLinkUp peut paramétrer ses propres notifications. L'adoption de ces fonctions de partage renforce la collaboration, la résilience et la prise de décision éclairée, pour un voyage plus autonome vers une meilleure santé.
