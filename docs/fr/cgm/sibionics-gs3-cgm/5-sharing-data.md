## **<u>5. Leçon 5</u>**

5.1 Partage des données avec votre professionnel médical
Appuyez sur l'icône Profil sur l'écran d'accueil de l'application. Sous Connexions,
appuyez sur Soins de santé pour inviter un professionnel de santé à consulter les
mesures de glycémie et les graphiques de tendance de votre capteur. Appuyez sur Se
connecter à une nouvelle clinique et saisissez l'identifiant de votre clinique.
En tant que professionnel de santé, vous pouvez créer un compte gratuit sur
SIBIONICS Proview via proview.sisensing.com/login. Vous pouvez vous inscrire en tant
que professionnel de santé pour accéder à la plateforme SIBIONICS ProView, puis
créer votre cabinet via le menu. Une fois votre cabinet créé, vous pourrez définir votre
identifiant de cabinet, que vous pourrez ensuite communiquer à vos patients.
En appuyant sur le logo SIBIONICS ProView, vous accédez à la page d'accueil où vous
pouvez voir vos patients. L'onglet Patient vous permet d'ajouter de nouveaux patients.
Dans votre liste de patients, vous pouvez gérer votre affichage pour ne voir que les
patients marqués d'une étoile ou ceux présentant un profil glycémique spécifique. Sur
la page d'accueil de SIBIONICS ProView, vous pouvez appuyer sur un patient
spécifique pour afficher ses données.

5.2 Partage de données avec votre famille
Au moment de la création de ce cours, le partage en temps réel des données GS3
avec les membres de la famille n'est pas encore disponible, mais il le sera bientôt.
Une fois cette fonctionnalité activée, vous pourrez sélectionner un compagnon
SIBIONICS dans l'onglet « Health Care Connections » (Connexions de soins de
santé) et inviter un abonné qui pourra surveiller vos données à distance.

5.3 Création de rapports
Les données quotidiennes indiquent la moyenne des mesures de glycémie
enregistrées par votre capteur, ainsi que votre temps dans la plage cible.

Parallèlement, la tendance quotidienne affiche les mesures quotidiennes de
glycémie sous forme de graphique, avec des icônes d'événements apparaissant aux
moments où elles ont été enregistrées.
Pour choisir la date d'un rapport, appuyez sur Précédent ou Suivant en haut de
l'écran pour parcourir les données quotidiennes, ou appuyez sur le bouton de date
pour sélectionner un jour spécifique à consulter.
Les rapports AGP résument les données sur plusieurs jours et montrent la
tendance et la variabilité des taux de glucose mesurés par le capteur au cours
d'une journée type. La bande verte dans l'AGP représente la plage de glucose
cible. Notez que l'AGP nécessite au moins 7 jours de mesures de glucose pour
garantir la validité statistique.
Pour exporter un rapport AGP, appuyez sur Rapport dans l'application GS3.
Définissez la période sur 7, 14, 30 ou 90 jours, ou appuyez sur Personnalisé pour
ajuster manuellement les dates de début et de fin. Appuyez ensuite sur Afficher le
rapport pour générer le rapport pour la période sélectionnée. Vous pouvez
télécharger ou partager le rapport à l'aide des icônes situées dans le coin supérieur
droit de l'écran.
Dans la plateforme SIBIOINCS ProView, il est possible de consulter 3 types de
rapports lorsque vous cliquez sur un patient depuis la page d'accueil : le rapport
AGP, le rapport comparatif et le rapport quotidien sur la glycémie. Pour chaque
rapport, il est possible de modifier la plage de dates. Dans le rapport comparatif,
vous pouvez comparer les détails AGP avec une période précédente. Dans le
rapport quotidien, vous pouvez passer la souris sur la courbe de glycémie pour voir
la valeur spécifique de glycémie, ainsi que les informations spécifiques enregistrées,
telles que les photos des aliments s'ils ont été pris à ce moment-là. Enfin, il est
toujours possible de télécharger les rapports à partir de l'icône de téléchargement
située dans le coin supérieur droit.
Dans cette vidéo, nous avons exploré les outils du CGM Sibionics GS3 permettant
de partager les données glycémiques et de générer des rapports. Grâce à ces
fonctionnalités, vous et votre équipe soignante pouvez collaborer plus efficacement,
ce qui vous permet de prendre des décisions opportunes, fondées sur des données,
pour améliorer votre santé.
