## **<u>4. Le Fonctionnement du système TouchCare Nano</u>**

Bienvenue dans cette leçon sur les fonctionnalités du TouchCare Nano System.
Dans ce guide, nous allons explorer comment utiliser les fonctionnalités du TouchCare
Nano System en mode automatique et en mode manuel, à la fois sur le PDM et sur
l'application EasyPatch.
4.1 Mode automatique
Le système TouchCare Nano ajuste automatiquement l'administration d'insuline en
temps réel, en fonction des lectures continues de la glycémie par le CGM.
Veuillez noter que le mode automatique et la gestion automatique des repas ne
peuvent être activés qu'à la suite d'une discussion avec votre professionnel de santé.
Une fois l'accord obtenu, un représentant de l'entreprise doit activer ces fonctions dans
le système.

Démarrage du mode automatique :
- Le mode Auto ne fonctionnera que si votre schéma basal et votre calculateur de
bolus ont déjà été configurés. Et n'oubliez pas qu'il ne peut pas être activé si
l'administration d'insuline est en pause.
- Pour lancer le mode automatique à partir du PDM, allez dans le menu principal,
appuyez sur EasyLoop, puis sur Auto Mode. Faites glisser l'interrupteur pour
l'activer. Il vous sera demandé d'entrer votre glycémie cible, votre poids et votre
dose quotidienne totale d'insuline. Lorsque le mode automatique est activé, le
menu Basal se transforme en menu Activité et une icône de bouclier vert
apparaît sur l'écran d'accueil. Si le système administre 30 % d'insuline en plus
ou en moins par rapport à votre débit basal habituel, une flèche apparaît sur le
bouclier. Si l'administration d'insuline est suspendue, une ligne rouge apparaît
en travers de l'icône. Si vous souhaitez à nouveau désactiver le mode
automatique, vous pouvez le faire à partir du même écran.
- Si vous utilisez l'application EasyPatch, allez dans le menu principal, touchez
EasyLoop, puis Auto Mode. Activez-le et entrez à nouveau votre objectif
glycémique, votre poids et votre dose totale d'insuline. Vous verrez la même
icône de bouclier vert une fois qu'elle sera active.
Démarrage de le traitement automatique des repas :
  - Le traitement automatique des repas est une fonction qui ajuste
automatiquement l'insuline lorsque vous saisissez que vous prenez un
repas comme le petit déjeuner, le déjeuner, le dîner ou une collation.
  - Pour l'activer à partir du PDM, allez au menu principal, appuyez sur
EasyLoop, puis sur Auto Mode. Dans cet écran, vous verrez une option
permettant d'activer le traitement automatique des repas. Vous aurez
besoin d'un mot de passe pour l'activer. Une fois qu'il est activé, vous
pouvez sélectionner le repas que vous prenez dans le menu Activité.
  - Dans l'application EasyPatch, allez dans le menu principal, appuyez sur
EasyLoop, puis sur Mode auto. Vous pouvez également activer le
traitement automatique des repas. Lorsqu'il est activé, l'icône de bolus
située en bas de votre écran d'accueil se transforme en icône d'activité. Il
vous suffit de la toucher pour annoncer votre repas.

Démarrage d'une activité sport :
  - Vous pouvez également lancer une activité sport qui réduira
l'administration d'insuline pendant l'activité physique.
  - Sur le PDM, accédez au menu principal et touchez Activité. Si le
traitement automatique des repas est activé, vous verrez des options
telles que le petit-déjeuner, le déjeuner, le dîner, le goûter et le Sport.
Touchez Sport et choisissez la durée de l'activité, de 30 minutes à 12
heures. Faites ensuite glisser pour confirmer.
  - Dans l'application EasyPatch, ouvrez le menu principal, appuyez sur
Événements, puis sur Sport, ou accédez au bouton de raccourci Activité
directement depuis l'écran d'accueil. Choisissez la durée de votre activité
et faites glisser pour la saisir.
Réglage des paramètres :
  - Les paramètres qui peuvent être ajustés pendant le mode automatique
sont la glycémie cible du capteur, le ratio de glucides et le facteur de
sensibilité à l'insuline.
  - Sur le PDM, vous pouvez modifier votre glycémie cible directement à
partir de l'écran du mode automatique en allant sur EasyLoop dans le
menu principal. Pour modifier votre ratio de glucides, allez dans
Réglages, puis Pompe à insuline, puis Configuration du bolus, et enfin
Configuration de la calculatrice de bolus.
  - Sur l'application EasyPatch, allez à EasyLoop dans le menu principal pour
ajuster votre objectif glycémique. Pour modifier votre ratio de glucides,
allez dans Réglages, puis Pompe à insuline, puis Configuration du
calculateur de bolus.
Administration du bolus :
  - Lors de l’administration d’un bolus, deux possibilités s'offrent à vous. Si
vous faites un comptage des glucides, vous pouvez utiliser le calculateur
de bolus. Sur le PDM, appuyez simplement sur Bolus dans le menu
principal et suivez les étapes. Si vous utilisez le traitement automatique
des repas, allez dans le menu Activité et sélectionnez votre repas.

○​ Dans l'application EasyPatch, appuyez sur l'icône bolus sur l'écran
d'accueil pour utiliser la calculatrice. Ou si la gestion automatique des
repas est activée, accédez au bouton de raccourci Activité sur l'écran
d'accueil et choisissez votre type de repas.
Suspendre l'administration d'insuline :
  - Il peut arriver que vous souhaitiez interrompre l'administration d'insuline.
Le système peut également le faire de lui-même s'il prévoit une
hypoglycémie.
  - Sur le PDM, touchez Suspendre dans le menu principal. Vous pouvez
également reprendre l'administration d'insuline au même endroit.
  - Dans l'application EasyPatch, allez dans Administration d'insuline à partir
du menu principal et appuyez sur Suspendre. Vous pourrez définir la
durée pendant laquelle vous souhaitez que l'insuline soit suspendue, puis
faire glisser pour confirmer. Le cercle entourant votre glycémie devient
rouge lorsque l'insuline est suspendue. Pour reprendre, revenez au même
écran et appuyez sur Reprendre.
4.2 Mode manuel et fonctions de sécurité
Lorsque le mode automatique est désactivé, le système fonctionne en mode manuel.
Dans ce mode, vous pouvez définir vos propres débits basaux ou utiliser des débits
basaux temporaires, et administrer des bolus d'insuline manuellement ou à l'aide du
calculateur de bolus.
Pour les repas riches en graisses ou consommés sur une longue période, vous pouvez
également utiliser les options de bolus prolongés ou combinés afin de mieux adapter
l'administration d'insuline à votre consommation alimentaire.
Le système comprend également une fonction de suspension prédictive en cas
d'hypoglycémie. Si votre taux de glucose baisse trop ou s'il est prévu qu'il baisse dans
les 30 prochaines minutes, le système suspendra automatiquement l'administration
d'insuline. Ce réglage peut être ajusté dans le menu EasyLoop. L'insuline reprendra
automatiquement lorsque les niveaux de glucose se stabiliseront, ou après une pause
maximale de deux heures.

En comprenant le fonctionnement du mode manuel et du mode automatique, ainsi que
les fonctions de sécurité intégrées, vous pouvez maximiser le système TouchCare Nano
et vous sentir plus confiant dans la gestion de votre diabète. Pour des étapes et des
réglages détaillés, reportez-vous toujours au manuel de votre appareil ou consultez
votre professionnel de santé.
