### 3.1 Transmetteur Dexcom G6 + Capteur Dexcom G6 + Pompe Tandem t:slim X2

Avec la pompe à insuline Tandem t:slim X2, la liaison d'un capteur est un processus simple. Voici comment le faire :

- Accédez à l’écran [Options] de la pompe, puis sélectionnez [Ma SGC].

- Choisissez le type de CGM que vous souhaitez utiliser.

- Si vous choisissez le Dexcom G6, vous devrez d'abord relier l'émetteur. Choisissez "Enter Transmitter ID" et suivez les instructions. La pompe vous demandera de confirmer l'ID de l'émetteur à deux reprises. La démarche de liaison est généralement transparente et sans problème. Veillez toutefois à ce que l'ancien émetteur ne se trouve pas à proximité du nouvel émetteur, faute de quoi Bluetooth pourrait détecter l'ancien émetteur. Éloignez-les d'au moins 6 mètres (20 pieds) pour éviter toute interférence. Une fois que vous avez réussi à relier le transmetteur à la pompe, vous pouvez démarrer le capteur :

- Dans le menu des options, sélectionnez à nouveau [Ma SGC].

- Choisissez [DÉMARRER CAPTEUR] et suivez les instructions.

- Au cours de cette étape, vous serez invité à saisir le code du capteur, qui se trouve sur le film protecteur du capteur. Si vous avez égaré le code du capteur, vous pouvez sélectionner [IGNORER]. Dans ce cas, la pompe vous demandera de calibrer le capteur après 2 heures, ce qui vous permettra de continuer à utiliser le capteur. Cependant, la pompe demandera une calibration toutes les 24 heures. Il est donc fortement conseillé de bien conserver le code du capteur pour éviter ces étapes supplémentaires. Si vous avez déjà démarré le capteur avec l'application Dexcom G6, la pompe récupère des données sans demander le code de capteur. Si vous utilisez un capteur Dexcom G7, vous pouvez sélectionner Dexcom G7 dans les options CGM. Vous verrez une option pour démarrer le capteur G7 sur la pompe, et vous serez invité à entrer le code d'appariement Dexcom G7 deux fois. Ce code d'appairage se trouve sur l'inserteur Dexcom G7. Si vous utilisez les capteurs FreeStyle Libre 2 ou 3 Plus, vous devrez démarrer votre capteur à l'aide de l'application Tandem t:slim mobile appariée plutôt qu'à l'aide de votre pompe. Pour commencer, appuyez sur Paramètres dans la barre de navigation de l'application et sélectionnez CGM, puis FreeStyle Libre 2 ou 3 Plus. Pour les utilisateurs d'iOS, appuyez sur “Démarrer le scan” ; les utilisateurs d'Android doivent s'assurer que la fonction NFC est activée dans les paramètres de leur téléphone. Tenez ensuite le dos de votre téléphone Android ou le haut de votre iPhone près du capteur jusqu'à ce qu'il vibre ou que vous entendiez une alerte. L'application Tandem t:slim mobile affiche l'écran “Session du capteur démarrée”. Appuyez sur Ok pour continuer. Une fois le capteur démarré, vous pouvez suivre la période d'échauffement sur l'application Tandem t:slim mobile et sur la pompe. En suivant ces étapes, vous pouvez relier un capteur Dexcom G6, G7 ou FreeStyle Libre 2 Plus à la pompe Tandem t:slim X2.

### 3.2 Comment relier l'application Tandem t:slim mobile à la pompe Tandem t:slim

X2 L'application Tandem t:slim mobile sert de compagnon à la pompe à insuline t:slim X2. Veuillez noter que la disponibilité et les fonctionnalités de l'application peuvent varier en fonction de votre lieu de résidence. Par exemple, depuis la dernière mise à jour de ce cours, la possibilité d'administrer un bolus d'insuline à partir de votre smartphone n'est disponible qu'aux États-Unis. Assurez-vous que votre smartphone est compatible avec l'application Tandem t:slim mobile Vous pouvez vérifier sa compatibilité sur le site web Tandem. L'utilisation d'un téléphone non pris en charge peut limiter certaines fonctionnalités de l'application. Après avoir téléchargé l'application Tandem t:slim mobile depuis le Play Store ou de l'App Store de votre appareil, localisez l'application sur votre smartphone et ouvrez-la. L'application vous demandera diverses autorisations et vous invitera à vous connecter ou à créer un compte Tandem Source. Si vous avez 13 ans ou plus, vous pouvez créer un compte personnel ou un compte de gardien. Sur l'écran d'accueil de votre pompe, appuyez sur “OPTIONS”, accédez à “Paramètres de l'appareil”, appuyez sur “Paramètres Bluetooth” et activez “Connexion mobile”. L'option “Appairer l'appareil” apparaît : appuyez dessus et une coche bleue génère un code d'appairage de l'appareil. Sur votre smartphone, appuyez sur “Commencer” pour démarrer le couplage avec la pompe, choisissez le numéro de série approprié de la pompe et saisissez le code de couplage de l'appareil dans l'application. Après avoir appuyé sur “Appairer avec la pompe” dans l'application, vous pouvez appuyer sur OK pour confirmer sur votre pompe. Vous pouvez maintenant appuyer sur Sync Pump Data (Synchroniser les données de la pompe) dans l'application Tandem t:slim mobile, et l'application affichera votre tableau de bord et commencera à afficher les données de la pompe. Gardez à l'esprit les recommandations suivantes :

- Pour garantir une communication ininterrompue entre votre pompe et l'application, il est recommandé de désactiver les paramètres d'optimisation de la batterie.

- L'application Tandem t:slim mobile doit fonctionner en arrière-plan pour transmettre et recevoir des données entre votre pompe et le nuage Tandem.

- Si vous souhaitez utiliser la fonction téléphone-bolus, vous devez activer les fonctions de sécurité de votre smartphone, telles que le verrouillage de l'écran, le code d'accès ou la reconnaissance faciale.

- Veillez

à désactiver l'automatisation des mises à jour du logiciel

d'exploitation de votre téléphone. Avant de mettre à jour manuellement le système d'exploitation de votre téléphone, assurez-vous que l'application Tandem t:slim mobile est compatible avec la nouvelle version du système d'exploitation.

- Lorsque des mises à jour de l'application Tandem t:slim mobile sont disponibles sur Google Play ou l'App Store, ne désinstallez pas l'application, mais mettez-la simplement à jour.

- Après avoir couplé et synchronisé votre pompe avec l'application Tandem t:slim mobile, vous pouvez vous attendre à utiliser environ 15 mégaoctets de données tous les trois jours. L'utilisation des données peut augmenter en cas de mauvaise réception cellulaire. Pour éviter d'utiliser les données cellulaires pour les téléchargements vers votre compte Tandem Source, réglez les paramètres de l'application pour ne télécharger que sur Wi-Fi en allant à to Settings to App to Data Control.

### 3.3 Comment créer un compte Tandem Source.

Les données de la pompe et du capteur peuvent être partagées avec un parent, un tuteur ou un prestataire de soins de santé via la plateforme Tandem Source.

- Les utilisateurs et les parents ou les soignants peuvent créer un compte Tandem Source via source.tandemdiabetes.com/login.

- Les utilisateurs de l'application Tandem t:slim mobile créent un compte Tandem Source lorsqu'ils se connectent pour la première fois à l'application.

- Les prestataires de soins de santé peuvent demander un compte professionnel Tandem Source par l'intermédiaire de leur représentant Tandem.

- Les utilisateurs administrateurs d'un compte clinique peuvent ajouter d'autres membres de l'équipe. Lorsque vous vous connectez à votre compte Tandem Source, vous trouverez plusieurs menus sur le côté gauche : Accueil, Rapports, Commandes, Ma pompe et Transférer la pompe. Dans le coin supérieur droit, vous pouvez accéder aux informations de votre compte Tandem et au menu d'aide. Vous pouvez télécharger manuellement les données de votre pompe en la connectant à votre ordinateur à l'aide d'un câble USB, ou vous pouvez utiliser l'application Tandem t:slim mobile qui synchronise automatiquement toutes les données de la pompe avec le Tandem Cloud environ une fois par heure. Tandem Source propose 3 rapports standard : le rapport d'ensemble, la chronologie quotidienne et les paramètres de la pompe. Via les paramètres du compte, vous pouvez partager les rapports avec votre prestataire de soins de santé.

### 3.4 Compte Glooko et téléchargement des données du capteur et de la pompe

Les données de la pompe et du capteur peuvent être partagées avec un professionnel médical via Glooko. Nous décrirons comment procéder en tant qu'utilisateur via Glooko Personal, et en tant que professionnel de la santé via Glooko for Professional Use. Glooko pour les diabétiques

- En tant qu'utilisateur, vous pouvez visiter my.glooko.com et créer votre compte.

- A un certain moment, vous serez en mesure d'introduire un "code proconnect" qui relie votre compte à votre hôpital.

- Créez un mot de passe pour votre compte et gardez-le accessible pour les connexions ultérieures, en particulier lorsque vous devez télécharger des données à partir de votre pompe. Lorsque vous vous connectez à Glooko, vous êtes invité à indiquer les appareils que vous utilisez. Pour la technologie Control-IQ, cela inclut la pompe Tandem t:slim X2 Vous devez ensuite télécharger le logiciel Glooko Uploader, qui est disponible via les paramètres dans le coin supérieur droit de l’interface. Une fois installé, vous trouverez un raccourci de Glooko Uploader sur votre bureau. Si vous ouvrez Glooko Uploader et connectez votre pompe Tandem à votre ordinateur à l'aide du câble USB fourni, les données de la pompe seront téléchargées sur votre ordinateur en quelques secondes. Lorsque vous ouvrez votre compte Glooko, vous trouverez les données téléchargées disponibles pour l’affichage. Glooko pour les professionnels de santé

- Les professionnels de santé ou les établissements médicaux peuvent obtenir un accès professionnel payant directement auprès de Glooko. Vous pouvez ensuite vous connecter via my.glooko.com en utilisant le nom d'utilisateur et le mot de passe fournis par votre établissement.

- Une fois connecté, vous trouverez un aperçu des derniers téléchargements de données et un onglet séparé affichant un aperçu de vos patients. Dans le coin supérieur droit, vous trouverez le code ProConnect de votre clinique.

- Vous pouvez personnaliser la liste des patients en la triant selon vos préférences ou en utilisant des drapeaux pour identifier facilement les patients dont les taux glycémiques sont élevés ou faibles. Vous pouvez ajouter un nouveau patient en cliquant sur le bouton situé dans le coin supérieur droit et vous pouvez ajouter son adresse e-mail pour l'inviter sur Glooko. Pour télécharger les données de la pompe et du capteur, les professionnels de santé peuvent soit acquérir le logiciel Glooko Uploader qui nécessite une licence par ordinateur, soit opter pour le dispositif transmetteur Glooko. De nombreuses cliniques préfèrent le transmetteur Glooko en raison de sa capacité de téléchargement rapide des dispositifs et de l'élimination des mises à jour fréquentes. Le transmetteur Glooko est compatible avec la plupart des dispositifs de traitement du diabète. Bien qu'il nécessite une connexion électrique, il transfère rapidement les données vers la plateforme cloud de Glooko en quelques secondes. Vous trouverez les données téléchargées dans l'onglet "Attribuer des appareils". Ici, vous pouvez attribuer les données au patient approprié. Si l'appareil a déjà été attribué à un patient spécifique, son nom sera suggéré. En suivant ces étapes, vous pourrez visionner les données de la pompe à insuline Tandem t:slim X2 dans Glooko, que ce soit pour un usage personnel ou en tant que professionnel de santé, et y accéder et les gérer de manière pratique. Les fournisseurs de soins de santé peuvent également accéder aux rapports des patients sans compte Glooko Pro si ces derniers partagent leurs identifiants de connexion.

### 3.5 Dexcom Follow en combinaison avec la technologie Tandem Control-IQ.

Si vous utilisez un capteur Dexcom avec la technologie Tandem Control-IQ, vous pouvez activer le suivi en temps réel des données de votre capteur pour les membres de votre famille ou les soignants grâce à l'application Dexcom G6 ou G7. Cette application vous permet de partager vos données avec un maximum de 10 soignants, à condition qu'ils téléchargent l'application Dexcom Follow. Des instructions de configuration détaillées sont disponibles dans le module dédié aux Dexcom G6 et G7. Il est important de noter que certains utilisateurs ont signalé des cas où la pompe Tandem perdait la connexion plus facilement lorsqu'ils utilisaient l'application Dexcom G6 ou G7. Si vous rencontrez ce problème, il est recommandé d'arrêter d'utiliser l'application Dexcom G6 ou G7 et d'utiliser l'application Dexcom dédiée. L’interconnexion des différents composants du système Tandem Control-IQ est une étape importante dans la mise en place du système automatisé d'administration d'insuline. Bien que le processus soit généralement simple et sans heurts, il peut arriver que vous rencontriez des défis ou des difficultés techniques. N'oubliez pas que vous n'êtes pas seul. Si vous rencontrez des problèmes ou si vous avez besoin d'aide, n'hésitez pas à demander du soutien. Vous pouvez rejoindre le groupe de soutien Facebook de la technologie Tandem Control-IQ ou entrer en contact avec d'autres utilisateurs du système. Ensemble, vous pouvez partager vos expériences, demander des conseils et trouver des solutions. Sachez que la résilience vient en cherchant de l'aide et en construisant une communauté de soutien autour de vous.
